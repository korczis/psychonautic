# AIAD Standard - Psychonautic

**Version**: 1.0.0
**Standard**: AI-Assisted Development (AIAD)
**Project**: Psychonautic Documentation Site

---

## Overview

AIAD provides structured AI collaboration through:
- **Agents**: Specialized AI personas for specific tasks
- **Commands**: Shorthand invocations for common operations
- **Doctrine**: Guiding principles and enforcement rules

---

## Directory Structure

```
.aiad/
├── README.md              # This file
├── agents/                # Agent definitions
│   ├── content-writer.agent.md
│   └── zola-specialist.agent.md
├── commands/              # Command definitions
│   ├── content.cmd.md
│   └── build.cmd.md
└── doctrine/              # Doctrine files
    └── no-mercy-no-doubts.doctrine.md
```

## Agents

| Agent | Domain | Invocation |
|-------|--------|------------|
| content-writer | Content | `/content` |
| zola-specialist | Build | `/build` |

---

## Commands

| Command | Description |
|---------|-------------|
| `/content` | Create, edit, or review content |
| `/build` | Build site, check links, deploy |

---

## Doctrine

**NO MERCY, NO DOUBTS** - Core operating principle

- NO MERCY: Complete execution, no placeholders, production-ready
- NO DOUBTS: Research first, verify facts, evidence-based

---

## Usage

Invoke agents by using their command in conversation:

```
/content - aktivovat content-writer agenta
/build - aktivovat zola-specialist agenta
```
