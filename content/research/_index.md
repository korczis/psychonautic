# Psychedelicky vyzkum - Renesance vedy

Psychedelicky vyzkum zaziva od pocatku 21. stoleti bezprecedentni **renesanci**. Po desetiletich prohibice a stigmatizace se prestizni akademicke instituce vracejí ke zkoumani terapeutickeho potencialu psychedelik. Tato sekce shrnuje soucasny stav vedeni, klicove instituce a slibné vysledky klinickych studii.

---

## Historicky kontext

### Chronologie psychedelickeho vyzkumu

| Obdobi | Charakteristika | Klicove udalosti |
|--------|-----------------|------------------|
| **1943-1965** | Zlata era | Objev LSD, prvni studie, tisice publikaci |
| **1966-2000** | Prohibice | Zarazení do Schedule I, zastavení vyzkumu |
| **2000-2010** | Opatrný navrat | Prvni FDA-schvalene studie |
| **2010-2020** | Renesance | Massive funding, Breakthrough Therapy |
| **2020-soucasnost** | Mainstreaming | Faze III studie, prvni schvaleni |

### Milniky moderni renesance

| Rok | Udalost | Vyznam |
|-----|---------|--------|
| **1990** | Strassmanova DMT studie | Prvni lidska studie po 20 letech |
| **2001** | MAPS zahajuje MDMA vyzkum | Systematicky terapeuticky vyzkum |
| **2006** | Johns Hopkins psilocybin studie | Mysticke zazitky u zdravych |
| **2011** | Robin Carhart-Harris fMRI studie | Neuroveda psychedelik |
| **2016** | Lancet: psilocybin pro depresi | Prulomove vysledky |
| **2017** | FDA Breakthrough Therapy (psilocybin) | Regulacni uznani |
| **2021** | MAPS MDMA faze III pozitivni | Blizko schvaleni |
| **2023** | Australie schvaluje MDMA/psilocybin | Prvni zeme s terapeutickym pristupem |

## Neuroveda psychedelik

### Default Mode Network (DMN)

DMN je sit mozkovych oblasti aktivních pri "odpočinku" a self-referenčním myseni.

```
DMN STRUKTURY:
==============
- Medialni prefrontalni kortex (mPFC)
- Posteriorni cingularni kortex (PCC)
- Angularni gyrus
- Lateralni temporalni kortex

NORMALNI STAV:
- Vysoka aktivita DMN
- Koherentni self-narativ
- Stabilni ego

POD PSYCHEDELIKY:
- Snizena aktivita DMN
- Disintegrace ega
- Zvysena entropie
- "Ego dissolution"
```

### Entropic Brain Hypothesis (Carhart-Harris)

```
SPEKTRUM VEDOMI:
================

Snízená entropie          Normální           Zvýšená entropie
        |                     |                     |
        v                     v                     v
+------------+         +------------+         +------------+
|  Rigidita  |         | Optimální  |         |   Chaos    |
|  Deprese   |         |  Zdravé    |         | Psychóza   |
|  OCD       |         |            |         |            |
| Zavislost  |         |            |         |            |
+------------+         +------------+         +------------+
        |                     |                     |
        +---------------------+---------------------+
                              |
                    PSYCHEDELIKA
                    zvyšují entropii
                    (kontrolované)
                              |
                              v
                    Terapeutické okno:
                    flexibilita bez chaosu
```

## Metodologie klinickych studii

### Typicky design psilocybinove studie

```
SCREENING (2-4 tydny pred)
├── Psychiatricke hodnoceni
├── Fyzicke vysetreni
├── Informovany souhlas
└── Pripravna sezeni (1-2)

DOZOVACI SEZENI
├── Den 0: Baseline mereni
├── Den 1: Psilocybin sezeni (6-8 hodin)
│   ├── Monitorovana mistnost
│   ├── 2 terapeuti
│   ├── Hudba, maska na oci
│   └── Minimalni verbální intervence
└── Den 2: Follow-up

INTEGRACE
├── Integracni sezeni (1-3)
├── Follow-up (1, 3, 6, 12 mesicu)
└── Dlouhodobe sledovani
```

### Klicove metodologicke aspekty

| Aspekt | Vyzva | Reseni |
|--------|-------|--------|
| **Zaslepení** | Ucinky jsou zjevne | Active placebo, nizka davka jako kontrola |
| **Expectancy effect** | Vysoka očekávání | Balanced design, mereni ocekavani |
| **Therapist effect** | Variabilita | Standardizovany protokol, trening |
| **Set & setting** | Non-farmakologicke faktory | Standardizace prostredi |

### Merici nastroje

| Nastroj | Meri | Pouziti |
|---------|------|---------|
| **MADRS/HAMD** | Deprese | Primarni endpoint |
| **CAPS-5** | PTSD | Primarni endpoint |
| **MEQ30** | Mysticka zkusenost | Prediktor outcome |
| **5D-ASC** | Zmenený stav | Fenomenologie |
| **EDI** | Ego disoluce | Proces |
| **CEQ** | Challenging experience | Bezpecnost |
