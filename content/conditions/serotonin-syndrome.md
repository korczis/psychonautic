# Serotoninovy syndrom - Serotonergni toxicita

**Serotoninovy syndrom** (serotonergni toxicita) je potencialne zivot ohrozujici stav zpusobeny nadbytkem serotoninu v centralnim nervovem systemu. Vznika typicky pri kombinaci serotonergních latek nebo predavkovani jednou z nich. Vcasna diagnostika a lecba jsou kriticke.

---

## Definice a epidemiologie

### Zakladni charakteristika

| Aspekt | Hodnota |
|--------|---------|
| **Definice** | Iatrogenni hyperserotonergni stav |
| **Pricina** | Kombinace/predavkovani serotonergik |
| **Nastup** | Minuty az hodiny po expozici |
| **Prubeh** | Rychla progrese mozna |
| **Mortalita** | 2-12% (pri zavaznych pripadech) |

### Epidemiologie

| Parametr | Hodnota |
|----------|---------|
| **Incidence** | Stoupajici (více serotonergnich léku) |
| **Nejcastejsi pricina** | SSRI + MAOi kombinace |
| **Podhlaseno** | Pravděpodobne značně |
| **Riziková populace** | Pacienti na kombinaci serotonergik |

## Priciny

### Mechanismus

```
SEROTONINOVY SYNDROM - MECHANISMUS

NORMALNI STAV:
Presynapticky     Synapse     Postsynapticky
neuron                         neuron
  |                              |
  |    5-HT                     5-HT receptory
  |    ------>                  (5-HT1A, 5-HT2A)
  |    <------
  |    (reuptake)
  |
  [MAO degradace]

SEROTONINOVY SYNDROM:
  |
  |    5-HT 5-HT 5-HT 5-HT
  |    ==================>      HYPERSTIMULACE
  |    X (blok reuptake)        receptoru
  |    X (MAO inhibice)
  |    + (zvysene uvolnění)
  |    + (prekurzory)

VYSLEDEK:
Masivni aktivace 5-HT1A a 5-HT2A
-> Autonomni, neuromuskularni, kognitivni symptomy
```

### Lekove skupiny

| Mechanismus | Priklady | Riziko |
|-------------|----------|--------|
| **MAO inhibitory** | Fenelzin, moklobemid, linezolid | Nejvyssi |
| **SSRI** | Fluoxetin, sertralin, paroxetin | Vysoke |
| **SNRI** | Venlafaxin, duloxetin | Vysoke |
| **TCA** | Amitriptylin, klomipramin | Stredni |
| **Triptany** | Sumatriptan | Nizke-stredni |
| **Opioidy** | Tramadol, fentanyl, meperidin | Stredni |
| **Prekurzory** | L-tryptofan, 5-HTP | Stredni |
| **Psychedelika** | MDMA, mescalin | Vysoke |
| **Jine** | Lithium, buspiron, DXM | Variabilni |

### Nebezpecne kombinace

| Kombinace | Riziko | Poznamka |
|-----------|--------|----------|
| **SSRI + MAOi** | **KRITICKE** | Absolutne kontraindikovano |
| **MDMA + MAOi** | **KRITICKE** | Potencialne fatalni |
| **MDMA + SSRI** | Vysoke | Snizeny ucinek + riziko |
| **Tramadol + SSRI** | Vysoke | Casto prehlizeno |
| **DXM + SSRI** | Vysoke | OTC riziko |
| **5-HTP + SSRI** | Stredni | Suplementy |
| **Triptany + SSRI** | Nizke-stredni | Kontroverzni |

### Psychedelika a serotonin syndrom

```
PSYCHEDELIKA A SS

MDMA (Extaze):
- SERT reversal -> masivni 5-HT uvolneni
- Kombinace s MAOi = FATALNI
- Kombinace s SSRI = Vysoke riziko
- Sam o sobe: Mirne riziko pri vysokych davkach

PSILOCYBIN, LSD:
- 5-HT2A AGONISTE (ne uvolneni)
- Nizke riziko SS pri izolovanem uziti
- ALE: Kombinace s MAOi/SSRI = Zvysene riziko

AYAHUASCA:
- DMT + MAOi (harmin)
- Inherentne serotonergnni kombinace
- Riziko pri kombinaci s SSRI!

DXM:
- SERT inhibitor + NMDA antagonista
- Vysoke riziko s SSRI
- FATALNI s MAOi
```

### Triada symptomu

```
SEROTONINOVY SYNDROM - TRIADA

1. AUTONOMNI DYSREGULACE
+---------------------------+
| - Tachykardie             |
| - Hypertenze (nebo labilni)|
| - Hyperthermie (>38C)     |
| - Poceni, flush           |
| - Mydriaza                |
| - Prujem                  |
+---------------------------+

2. NEUROMUSKULÁRNÍ ABNORMALITY
+---------------------------+
| - Tremor                  |

| - Klonus (spontanni,      |
|   indukovatelny)          |
| - Rigidita                |
| - Myoklonus               |

+---------------------------+

3. ZMENENY MENTALNÍ STAV
+---------------------------+
| - Agitace                 |
| - Uzkost                  |
| - Zmatenost               |
```

### Zavaznost

| Stupen | Symptomy | Management |
|--------|----------|------------|
| **Mirny** | Tremor, tachykardie, mydriaza | Observace, vysazeni leku |
| **Stredni** | Hyperthermie, hyperreflexie, agitace | Hospitalizace, benzodiazepiny |
| **Tezky** | Hyperthermie >40C, rigidita, koma | ICU, intubace, cyproheptadin |

## Diagnostika

### Hunter Criteria (nejpresnejsi)

Pozitivni pokud je pritomen **serotonergni agens** + **jeden z nasledujicich**:

| Kriterium | Popis |
|-----------|-------|
| **1** | Spontanni klonus |
| **2** | Indukovatelny klonus + (agitace NEBO diaforizace) |
| **3** | Okulární klonus + (agitace NEBO diaforizace) |
| **4** | Tremor + hyperreflexie |
| **5** | Hypertonie + teplota >38C + (okulární NEBO indukovatelny klonus) |

### Diferencialni diagnostika

| Stav | Odlisnosti |
|------|------------|
| **Neurolepticky maligní syndrom** | Pomaly nastup (dny), "olověná" rigidita, antipsychotika |
| **Maligní hyperthermie** | Anesteticka latka, svalová nekroza |
| **Anticholinergni toxicita** | Sucha kuze, retence, ileus, SNIZENE strevni zvuky |
| **Sympatomimeticka intoxikace** | Podobne, ale bez klonus, bez hyperreflexie |
| **Infekce CNS** | Meningeální příznaky, zánětlivé markery |

### Laboratorni nalezy

| Test | Nalez |
|------|-------|
| **CK** | Zvysena (myolyza) |
| **Leukocyty** | Mirna leukocytoza |
| **Trombocyty** | Mozna trombocytopenie (DIC) |
| **Metabolická acidóza** | Pri tezke hyperthermii |
| **Jaterní enzymy** | Mohou byt zvyseny |
| **Renální funkce** | Mozna AKI (rabdomyolyza) |

### Farmakoterapie

| Lek | Indikace | Davkovani |
|-----|----------|-----------|
| **Benzodiazepiny** | Agitace, prevence krecí | Diazepam 5-10 mg IV |
| **Cyproheptadin** | Specificka antiserotoninergni | 12 mg p.o., pak 2 mg q2h |
| **Chlorpromazin** | Alternativa cyproheptadinu | 50-100 mg IM (opatrne - hypotenze) |
| **Dantrolenium** | Tezka rigidita | 2.5 mg/kg IV |

### Co NEPODAVAT

| Kontraindikovane | Duvod |
|------------------|-------|
| **Dalsi serotonergika** | Zhorseni |
| **Bromokriptin** | Neefektivni (to je pro NMS) |
| **Antipyretika** | Neefektivni (neni cytokinová horečka) |

## Prevence

### Rizikove situace

| Situace | Prevence |
|---------|----------|
| **Zmena antidepresiva** | Wash-out perioda |
| **Pridani druheho serotonergika** | Zvazit nutnost |
| **Suplementy (5-HTP)** | Informovat pacienty |
| **OTC leky (DXM)** | Screening |
| **Rekreacni drogy (MDMA)** | Harm reduction edukace |

### Wash-out periody

| Lek | Washout pred MAOi |
|-----|-------------------|
| **Fluoxetin** | 5-6 tydnu (dlouhy T1/2) |
| **Paroxetin** | 2 tydny |
| **Sertralin** | 2 tydny |
| **Venlafaxin** | 2 tydny |
| **MAOi pred SSRI** | 2 tydny |

### Harm reduction u psychedelik

| Latka | Doporuceni |
|-------|------------|
| **MDMA** | NIKDY s MAOi; minimalne 2 tydny po SSRI |
| **Ayahuasca** | Obsahuje MAOi - vysadit SSRI minimalne 2-4 tydny pred |
| **Psilocybin** | Nizke riziko, ale opatrnost s SSRI |
| **5-HTP suplementy** | Nepouzivat s SSRI |

## Kazuistiky

### Pripad 1: MDMA + MAOi

> 24-lety muz uzil MDMA na party. Nevedel, ze jeho "herbalní" antidepresivum obsahuje MAOi (moclobemid analog). Behem 2 hodin: teplota 41C, klonus, tachykardie 150/min, zmatenost. ICU, intubace, chlazeni, cyproheptadin. Přežil s reziduálními kognitivnimi deficity.

### Pripad 2: DXM + SSRI

> 19-leta zena na fluoxetinu uzila vysokou davku DXM (pro rekreacni ucinek). Behem 4 hodin: agitace, tremor, hyperreflexie, teplota 38.5C. Hospitalizace, benzodiazepiny, vysazení fluoxetinu. Rezoluce za 48 hodin.
