# Nukleotidy

Nukleotidy jsou základní stavební jednotky nukleových kyselin a klíčové molekuly v energetickém metabolismu.

## Adenosinové nukleotidy

| Nukleotid | Funkce |
|-----------|--------|
| [ATP](/nucleotids/atp/) | Energetická měna |
| [ADP](/nucleotids/adp/) | Substrát pro fosforylaci |
| [AMP](/nucleotids/amp/) | Signální molekula |
| [cAMP](/nucleotids/camp/) | Druhý posel |

## Guanosinové nukleotidy

| Nukleotid | Funkce |
|-----------|--------|
| [GTP](/nucleotids/gtp/) | G-proteiny |
| [GDP](/nucleotids/gdp/) | Substrát |
| [GMP](/nucleotids/gmp/) | Metabolit |
