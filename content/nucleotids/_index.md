# Nukleotidy v neurofarmakologii

Nukleotidy jsou fundamentalni molekuly zivota, ktere daleko presahuji svou znamou roli stavebnich jednotek nukleovych kyselin. V centralni nervove soustave (CNS) plni nukleotidy klicove funkce v energetickem metabolismu, neurotransmisi a intracelularni signalizaci. Jejich vliv na neuronalni funkce je cilem psychoaktivnich latek a predstavuje slibnou oblast pro vyvoj novych farmak.

---

## Uvod do neurofarmakologie nukleotidu

### Historicky kontext

Objev role nukleotidu v nervovem systemu predstavuje jednu z nejvyznamnejsich kapitol moderni neurovedecke historie:

| Rok | Objev | Vyzkumnik | Vyznam |
|-----|-------|-----------|--------|
| 1929 | Izolace ATP | Karl Lohmann | Identifikace "energeticke meny" |
| 1956 | Objev cAMP | Earl Sutherland | Koncept druheho posla |
| 1963 | cAMP jako druhy posel | Sutherland & Rall | Nobelova cena 1971 |
| 1970 | Purinergni neurotransmise | Geoffrey Burnstock | ATP jako neurotransmiter |
| 1972 | cGMP v nervovem systemu | Ferid Murad | Nobelova cena 1998 |
| 1978 | P2 purinergni receptory | Burnstock | Klasifikace P2X/P2Y |
| 1994 | Klonovani P2X receptoru | Gary Bhuen | Molekularni identifikace |
| 2012 | Krystalova struktura P2X | Kawate et al. | Strukturni biologie |

### Vyznam pro psychonauiku

Nukleotidy a jejich receptory predstavuji dulezite, avsak casto prehlizene, cile psychoaktivnich latek:

1. **Energeticky metabolismus** - Psychedelika moduluji neuronalni energetiku
2. **Signalni kaskady** - cAMP/cGMP zprostredkovavaji ucinky mnoha latek
3. **Neuromodulace** - ATP jako ko-transmiter ovlivnuje synaptickou plasticitu
4. **Neuroprotekce** - Adenosin ma protektivni ucinky v CNS

## ATP jako bunecna energeticka mena

### Termodynamika ATP hydrolzy

ATP je univerzalni "energeticka mena" zivych organismu. Hydrolza ATP uvolnuje energii, ktera pohani vetisnu bunecnych procesu:

```
ATP + H2O -> ADP + Pi + ENERGIE

Standardni volna energie (DeltaG°'):
ATP -> ADP + Pi:    -30.5 kJ/mol (-7.3 kcal/mol)
ATP -> AMP + PPi:   -45.6 kJ/mol (-10.9 kcal/mol)

FYZIOLOGICKE PODMINKY (DeltaG):
ATP -> ADP + Pi:    -50 az -65 kJ/mol
(vyrazne negativnejsi nez standardni hodnota)

PRICINY:
1. [ATP] >> [ADP] v buuce (pomere ~10:1)
2. [Pi] relativne nizka
3. Mg2+ komplexace
```

### ATP v neuronech

Neurony maji extrerne vysoke energeticke naroky:

| Proces | Spotreba ATP | Podil (%) |
|--------|--------------|-----------|
| Na+/K+-ATPaza | 40-60% | Udrzba klidoveho potencialu |
| Ca2+-ATPaza | 15-20% | Regulace [Ca2+]i |
| Synapticky cyklus | 15-25% | Uvolnovani a recyklace vezikul |
| Synteza proteinu | 5-10% | Plasticita, opravy |
| Axonalni transport | 5-10% | Distribuce organil |

```
NEURONALNI ENERGETICKY ROZPOCET

                Na+/K+-ATPaza (50%)
               ___________________
              |                   |
              |    NEURON         |
              |                   |
Ca2+-ATPaza   |   [ATP] = 3-5 mM |  Synapticky
   (18%)      |   [ADP] = 0.3 mM |  cyklus (20%)
              |   ATP/ADP ~ 10:1 |
              |___________________|
                        |
                  Ostatni (12%)
```

## Purinergni signalizace v CNS

### ATP jako neurotransmiter

Geoffrey Burnstock v roce 1970 navrd koncept purinergni neurotransmise. ATP funguje jako:

1. **Primy neurotransmiter** - uvolnovan ze synaptickych vezikul
2. **Ko-transmiter** - spolupusobi s klasickymi neurotransmitery
3. **Signalni molekula** - uvolnovana pri bunecnem poskozeni

| Nukleotid | Funkce | Signalni role |
|-----------|--------|---------------|
| [ATP](/nucleotids/atp/) | Energeticka mena, neurotransmiter | P2X/P2Y aktivace |
| [ADP](/nucleotids/adp/) | Substat pro fosforylaci | P2Y1/P2Y12 signalizace |
| [AMP](/nucleotids/amp/) | Signalni molekula | AMPK aktivace |
| [cAMP](/nucleotids/camp/) | Druhy posel | PKA/CREB/Epac |

### Guanosinove nukleotidy

| Nukleotid | Funkce | Signalni role |
|-----------|--------|---------------|
| [GTP](/nucleotids/gtp/) | G-proteinova signalizace | Molekularni prepinac |
| [GDP](/nucleotids/gdp/) | G-proteinovy cyklus | Inaktivni stav G-proteinu |
| [GMP](/nucleotids/gmp/) | Metabolit | Prekurzor cGMP |

### Cytidinove nukleotidy

| Nukleotid | Funkce |
|-----------|--------|
| [CTP](/nucleotids/ctp/) | Synteza fosfolipidu |
| [CDP](/nucleotids/cdp/) | Lipidovy metabolismus |
| [CMP](/nucleotids/cmp/) | Metabolit |

### Uridinove nukleotidy

| Nukleotid | Funkce |
|-----------|--------|
| [UTP](/nucleotids/utp/) | RNA synteza, P2Y signalizace |
| [UMP](/nucleotids/ump/) | Metabolit |

### Thymidinové nukleotidy

| Nukleotid | Funkce |
|-----------|--------|
| [TTP](/nucleotids/ttp/) | DNA synteza |
| [TDP](/nucleotids/tdp/) | Kofaktor enzymu |

## Metodologie vyzkumu

### Experimentalni pristupy

| Metoda | Aplikace | Omezeni |
|--------|----------|---------|
| Radioligand binding | Receptorova charakterizace | In vitro pouze |
| FRET/BRET biosenzory | cAMP/cGMP dynamika v realnem case | Exprese konstruktu |
| Patch clamp | P2X aktivita | Nizka propustnost |
| Mikrodialyza | Extracellularni purinovy | Casove rozliseni |
| PET imaging | In vivo receptory | Radiotracery |
| Optogenetika | Casoprostorova kontrola | Geneticka modifikace |

### Farmakologické nástroje

| Nastroj | Cil | Pouziti |
|---------|-----|---------|
| NECA | A1/A2 agonista | Neselektivni aktivace |
| DPCPX | A1 antagonista | Selektivni blokada |
| CGS21680 | A2A agonista | Selektivni aktivace |
| BzATP | P2X7 agonista | Aktivace P2X7 |
| Suramin | P2 antagonista | Neselektivni blokada |
| Forskolin | Aktivator AC | Zvyseni cAMP |
| IBMX | PDE inhibitor | Zabraneni degradace cAMP/cGMP |
