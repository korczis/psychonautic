# Psilocybes - Lysohlávky

Psilocybes (lysohlávky) jsou rodem hub obsahujících **psychoaktivní tryptaminy**, především [psilocybin](/alkaloids/psilocybin/) a [psilocin](/alkaloids/psilocin/).

## Aktivní látky

| Látka | Typ | Receptor |
|-------|-----|----------|
| [Psilocybin](/alkaloids/psilocybin/) | Prodrug | 5-HT2A |
| [Psilocin](/alkaloids/psilocin/) | Aktivní metabolit | 5-HT2A |
| [Baeocystin](/alkaloids/baeocystin/) | Minor alkaloid | 5-HT2A |
| [Norbaeocystin](/alkaloids/norbaeocystin/) | Minor alkaloid | 5-HT2A |

## Biochemie

Psilocybin je **fosforylovaný derivát** tryptaminu. Po požití je defosforylován na psilocin, který působí jako agonista serotoninových receptorů.

```
Psilocybin → [Alkalická fosfatáza] → Psilocin → 5-HT2A receptor
```

## Právní status

V České republice jsou psilocybin a psilocin zařazeny na seznam zakázaných látek (Nařízení vlády č. 463/2013 Sb.).
