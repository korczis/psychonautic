# Psilocybe cubensis - Kubánská lysohlávka

**Psilocybe cubensis** je nejznámější a nejčastěji kultivovaný druh [psilocybinových hub](/psilocybes/). Pochází z tropických a subtropických oblastí.

## Taxonomie

| Klasifikace | Hodnota |
|-------------|---------|
| Říše | Fungi |
| Oddělení | Basidiomycota |
| Třída | Agaricomycetes |
| Řád | Agaricales |
| Čeleď | Hymenogastraceae |
| Rod | [Psilocybe](/psilocybes/) |
| Druh | P. cubensis |

## Morfologie

### Klobouk
- Průměr: 2-8 cm
- Tvar: konvexní až plochý
- Barva: zlatohnědá až karamelová
- Povrch: hladký, lepkavý za vlhka

### Třeň
- Výška: 4-15 cm
- Průměr: 0,5-1,5 cm
- Barva: bílá až nažloutlá
- Modření: silné při poškození

### Výtrusy
- Barva: tmavě purpurově hnědá
- Rozměry: 11-17 × 7-12 μm
- Tvar: elipsoidní

## Aktivní látky

| Alkaloid | Koncentrace (suchá hmotnost) |
|----------|------------------------------|
| [Psilocybin](/alkaloids/psilocybin/) | 0,14-0,42% |
| [Psilocin](/alkaloids/psilocin/) | 0,01-0,17% |

## Biotop

- **Substrát**: Trus býložravců (krávy, koně, sloni)
- **Klima**: Tropické a subtropické
- **Rozšíření**: Střední a Jižní Amerika, JV Asie, Austrálie

## Odrůdy (Strains)

Existuje mnoho kultivarů s různými charakteristikami:

| Odrůda | Charakteristika |
|--------|-----------------|
| Golden Teacher | Střední potence, vhodná pro začátečníky |
| B+ | Velké plodnice, robustní růst |
| Penis Envy | Vysoká potence, mutantní morfologie |
| Albino A+ | Albinistická forma, bílé plodnice |
| Ecuador | Tropický původ, odolný |

## Biochemický mechanismus

Psilocybin je v těle metabolizován na psilocin:

```
Psilocybin (O-fosforyl-4-hydroxy-DMT)
    ↓ [Alkalická fosfatáza]
Psilocin (4-hydroxy-DMT)
    ↓
5-HT2A receptor → Psychoaktivní účinky
```

Viz také: [Metabolismus psilocybinu](/alkaloids/psilocybin/#metabolismus)
