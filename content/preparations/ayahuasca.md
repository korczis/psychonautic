# Ayahuasca - Svaty napoj Amazonie

**Ayahuasca** (z kecuastiny *aya* = duse, *waska* = liana) je tradicni psychoaktivni napoj puvudem z Amazonskeho deštneho pralesa. Pripravuje se kombinaci dvou rostlin - liany **Banisteriopsis caapi** obsahujici [MAO inhibitory](@/pharmacology/maoi.md) a listu **Psychotria viridis** obsahujicich [DMT](@/alkaloids/dmt.md). Tato synergicka kombinace umoznuje oralni aktivitu DMT a poskytuje 4-6 hodinovy intenzivni psychedelicky zazitek.

---

## Zakladni charakteristika

| Vlastnost | Hodnota |
|-----------|---------|
| **Nazvy** | Ayahuasca, Yage, Caapi, Daime, Hoasca, La Purga |
| **Typ** | Psychoaktivni napoj |
| **Puvod** | Amazonie (Peru, Brazilie, Kolumbie, Ekvador) |
| **Stari tradice** | 1000-5000+ let |
| **Hlavni slozky** | B. caapi (MAOI) + P. viridis (DMT) |
| **Mechanismus** | MAO inhibice → oralni aktivita DMT |
| **Trvani** | 4-6 hodin |
| **Legal status** | Variabilni (nabozenske vyjimky) |

## Priprava ayahuascy

### Tradicni metoda

```
SUROVINY:
- B. caapi (rozmlacena lodyha): 50-100g
- P. viridis (cerstve listy): 25-50g
- Voda: ~3-5 litru

POSTUP:
1. Rozmlaceni caapi na vlakna
2. Vrstevní caapi a chacruna do hrnce
3. Pridani vody
4. Vareni 4-12 hodin na mirnem ohni
5. Scezeni tekutiny
6. Opakovaní vareni (2-3x)
7. Kombinace vyvarů
8. Redukovani na pozadovany objem (~100-200 mL/davku)
```

### Davkovani

| Uroven | B. caapi (sucha) | P. viridis | DMT odhad |
|--------|------------------|------------|-----------|
| **Lehka** | 30-50g | 15-25g | 20-40 mg |
| **Stredni** | 50-80g | 25-40g | 40-70 mg |
| **Silna** | 80-120g | 40-60g | 70-100 mg |

**Poznamka**: Potence velmi variabilni - zaleži na kvalite materialu.
