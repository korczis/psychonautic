# Visual Network (VN) - Vizuální síť

**Visual Network** (VN, "vizuální síť") je rozsáhlá mozková síť specializovaná na **zpracování vizuálních informací** od základní detekce světla po komplexní rozpoznávání objektů. Pod vlivem [psychedelik](@/glossary/psychedelikum.md) vykazuje VN dramatické změny aktivity, které se projevují jako **vizuální halucinace**, **geometrické vzory** a **změněná percepce barev**.

---

## Základní informace

| Aspekt | Popis |
|--------|-------|
| **Definice** | Hierarchická mozková síť zpracovávající vizuální informace |
| **Objevena** | Hubel & Wiesel (1960s) - visual cortex organization |
| **Aktivita** | Kontinuální během zrakového vnímání |
| **Funkce** | Edge detection, color processing, object recognition, motion |
| **Klinický význam** | Alterace u migrény, epilepsie, psychedelické stavy |

## VN a psychedelika

### Vizuální efekty

| Efekt | Mechanismus | Typické látky |
|-------|-------------|---------------|
| **Geometrické vzory** | Spontánní V1 aktivita | Všechna psychedelika |
| **Color enhancement** | V4 hyperaktivita | Mescalin, 2C-B |
| **Object morphing** | Altered ventral stream | LSD, psilocybin |
| **Trails/tracers** | Motion processing | LSD |
| **Fractální vzory** | Recursive processing | DMT, LSD |
