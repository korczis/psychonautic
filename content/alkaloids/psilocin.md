# Psilocin - 4-hydroxy-N,N-dimethyltryptamin

**Psilocin** (4-HO-DMT) je psychoaktivní alkaloid a **aktivní metabolit** [psilocybinu](/alkaloids/psilocybin/). Na rozdíl od psilocybinu působí **přímo** na serotoninové receptory bez nutnosti metabolické aktivace.

## Chemická struktura

### Základní parametry
| Vlastnost | Hodnota |
|-----------|---------|
| **Chemický název** | 4-hydroxy-N,N-dimethyltryptamin |
| **Alternativní názvy** | 4-HO-DMT, Psilocyn |
| **Molekulární vzorec** | C₁₂H₁₆N₂O |
| **Molekulární hmotnost** | 204,27 g/mol |
| **CAS číslo** | 520-53-6 |
| **IUPAC** | 3-[2-(dimethylamino)ethyl]-1H-indol-4-ol |

### Strukturní srovnání

| Sloučenina | Struktura | Aktivita |
|------------|-----------|----------|
| **Psilocin** | 4-HO-DMT | Přímo aktivní |
| [Psilocybin](/alkaloids/psilocybin/) | 4-PO-DMT | Prodrug |
| DMT | N,N-DMT | Přímo aktivní |
| Serotonin | 5-HT | Endogenní |

### Vztah k serotoninu

```
Serotonin (5-HT):     HO-[Indol]-CH₂-CH₂-NH₂
                           ↑5
Psilocin (4-HO-DMT):  HO-[Indol]-CH₂-CH₂-N(CH₃)₂
                           ↑4
```

Psilocin je strukturně podobný serotoninu, ale:
- Hydroxylová skupina na pozici **4** místo 5
- **N,N-dimethylace** terminálního aminu

## Fyzikálně-chemické vlastnosti

| Vlastnost | Hodnota |
|-----------|---------|
| **Teplota tání** | 173-176°C |
| **Barva** | Bílá → modrá (oxidace) |
| **Rozpustnost** | Rozpustný v methanolu, ethanolu |
| **Stabilita** | **Nestabilní** - rychle oxiduje |
| **pKa** | 8,5 (amino skupina) |

### Modření (Bluing Reaction)

Psilocin je zodpovědný za charakteristické **modření** hub při poškození:

```
Psilocin → [O₂, oxidázy] → Psilocin-o-chinon (modrý polymer)
```

- **Mechanismus**: Oxidace 4-hydroxyindolu
- **Katalyzátory**: Vzdušný kyslík, lakkázy
- **Produkt**: Modro-zelený pigment (dimery/polymery)

**Poznámka**: Modření ≠ potence, ale indikuje přítomnost 4-substituovaných tryptaminů.

## Farmakologie

### Receptorový profil

| Receptor | Ki (nM) | Účinek |
|----------|---------|--------|
| **5-HT2A** | 6-15 | **Plný agonista** (psychoaktivita) |
| **5-HT2B** | 4-10 | Agonista |
| **5-HT2C** | 10-20 | Agonista (modulace nálady) |
| **5-HT1A** | 50-100 | Parciální agonista (anxiolýza) |
| **D1** | >1000 | Minimální |

### Srovnání s jinými tryptaminy

| Látka | 5-HT2A Ki | Relativní potence |
|-------|-----------|-------------------|
| LSD | 1-2 nM | Nejvyšší |
| **Psilocin** | 6-15 nM | Vysoká |
| DMT | 75-130 nM | Střední |
| Serotonin | 300-500 nM | Nízká |

### Mechanismus účinku

```
Psilocin
    ↓
5-HT2A receptor (kortex)
    ↓
Gq/11 protein aktivace
    ↓
Fosfolipáza C (PLC)
    ↓
IP₃ + DAG
    ↓
Ca²⁺ release + PKC aktivace
    ↓
Změna neuronální aktivity
    ↓
Psychoaktivní účinky
```

### Neurobiologie

#### Default Mode Network (DMN)
- **Efekt**: Snížení konektivity v DMN
- **Výsledek**: Ego dissolution, mystické zážitky

#### Claustrum
- **Efekt**: Změna integrace senzorických dat
- **Výsledek**: Synestézie, změněné vnímání

#### Thalamo-kortikální okruhy
- **Efekt**: Změna filtrování informací
- **Výsledek**: Rozšíření vědomí

## Farmakokinetika

### Absorpce a distribuce

| Parametr | Hodnota |
|----------|---------|
| **Biologická dostupnost (p.o.)** | 50-60% |
| **Tmax** | 30-60 minut |
| **Vd** | ~1 L/kg |
| **Vazba na proteiny** | Nízká (~15%) |
| **Průchod BBB** | Ano (lipofilní) |

### Metabolismus

```
Psilocin
    ↓
┌───────────────────────────────────────┐
│ Fáze I                                │
├───────────────────────────────────────┤
│ [MAO-A] → 4-hydroxyindol-3-acetaldehyd│
│ [Oxidázy] → Psilocin-chinon           │
└───────────────────────────────────────┘
    ↓
┌───────────────────────────────────────┐
│ Fáze II                               │
├───────────────────────────────────────┤
│ [UGT1A10] → Psilocin-O-glukuronid     │
│ [SULT] → Psilocin-O-sulfát            │
└───────────────────────────────────────┘
    ↓
Renální eliminace (80%)
```

### Eliminace

| Parametr | Hodnota |
|----------|---------|
| **T½ eliminace** | 2,5-3 hodiny |
| **Clearance** | ~1,5 L/h/kg |
| **Hlavní cesta** | Glukuronidace (UGT1A10) |
| **Moč** | 80% (konjugáty) |

## Výskyt v houbách

### Koncentrace podle druhů

| Druh | Psilocin (% sušiny) |
|------|---------------------|
| P. azurescens | 0,38% |
| P. bohemica | 0,11% |
| [P. cubensis](/shrooms/psilocybes/cubensis/) | 0,01-0,17% |
| [P. semilanceata](/shrooms/psilocybes/semilanceata/) | 0,02% |

**Poznámka**: Většina hub obsahuje více [psilocybinu](/alkaloids/psilocybin/) než psilocinu kvůli nestabilitě psilocinu.

### Degradace v houbách

```
Čerstvé houby: Psilocybin >> Psilocin
       ↓ [Sušení, stárnutí, oxidace]
Staré houby:   Psilocybin > Psilocin → Degradační produkty
```

## Syntéza (laboratorní)

### Hofmannova syntéza (1958)

```
4-Hydroxyindol
    ↓ [Oxalyl chlorid]
4-Hydroxyindol-3-glyoxylyl chlorid
    ↓ [Dimethylamin]
4-Hydroxy-N,N-dimethyltryptamin (Psilocin)
```

### Moderní přístupy
- **Biosyntéza**: Enzymatická produkce v E. coli
- **Chemoenzymatická**: Kombinace chemie a enzymů
- **Fermentace**: Geneticky modifikované houby

### Ehrlichův test

```
Psilocin + p-dimethylaminobenzaldehyd → Fialové zbarvení
```

- **Reagent**: Ehrlichův (p-DMAB v HCl)
- **Pozitivní**: Fialová/purpurová barva
- **Specifita**: Indoly obecně (nespecifický)

## Srovnání: Psilocin vs Psilocybin

| Vlastnost | Psilocin | [Psilocybin](/alkaloids/psilocybin/) |
|-----------|----------|------------|
| **Aktivita** | Přímo aktivní | Prodrug |
| **Stabilita** | Nestabilní | Stabilní |
| **Rozpustnost** | Lipofilní | Hydrofilní |
| **Průchod BBB** | Rychlý | Po konverzi |
| **Nástup účinku** | Rychlejší | Pomalejší |
| **Výskyt v houbách** | Menší | Větší |
