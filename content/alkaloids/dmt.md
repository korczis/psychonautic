# DMT - N,N-Dimethyltryptamin

**DMT** (N,N-Dimethyltryptamin) je mocný psychedelický tryptamin známý jako **"Spirit Molecule"** (Molekula ducha). Je to **endogenní sloučenina** - přirozeně se vyskytuje v lidském těle, rostlinách i zvířatech. DMT je aktivní složkou ayahuascy a poskytuje intenzivní, ale krátké psychedelické zážitky.

## Chemická struktura

### Základní parametry

| Vlastnost | Hodnota |
|-----------|---------|
| **Chemický název** | N,N-Dimethyltryptamin |
| **Alternativní názvy** | Dimitri, Spirit Molecule, Businessman's Trip |
| **Molekulární vzorec** | C₁₂H₁₆N₂ |
| **Molekulární hmotnost** | 188,27 g/mol |
| **CAS číslo** | 61-50-7 |
| **IUPAC** | 2-(1H-indol-3-yl)-N,N-dimethylethanamine |

### Strukturní rodina

| Sloučenina | Struktura | Aktivita |
|------------|-----------|----------|
| **DMT** | N,N-DMT | Přímo aktivní |
| 5-MeO-DMT | 5-methoxy-DMT | Silnější |
| [Psilocin](/alkaloids/psilocin/) | 4-HO-DMT | Přímo aktivní |
| [Psilocybin](/alkaloids/psilocybin/) | 4-PO-DMT | Prodrug |
| Bufotenin | 5-HO-DMT | Slabší |

### Srovnání se serotoninem

```
Serotonin (5-HT):    HO-[Indol]-CH₂-CH₂-NH₂
                          ↑5
DMT:                    [Indol]-CH₂-CH₂-N(CH₃)₂

Psilocin (4-HO-DMT): HO-[Indol]-CH₂-CH₂-N(CH₃)₂
                          ↑4
```

## Endogenní DMT

### Výskyt v lidském těle

DMT je **endogenní neurotransmiter**:

| Tkáň/Tekutina | Přítomnost |
|---------------|------------|
| Mozek | ✅ Potvrzeno |
| Cerebrospinální tekutina | ✅ Potvrzeno |
| Krev | ✅ Stopy |
| Moč | ✅ Stopy |
| Epifýza (šišinka) | Spekulace |

### Biosyntéza v těle

```
L-Tryptofan
    ↓ [AADC - aromatická dekarboxyláza]
Tryptamin
    ↓ [INMT - indolethylamin-N-methyltransferáza]
N-Methyltryptamin (NMT)
    ↓ [INMT]
N,N-Dimethyltryptamin (DMT)
```

| Hypotéza | Status |

| Snění/REM spánek | Spekulace |
| Zážitky blízké smrti (NDE) | Spekulace |

## Farmakologie

### Receptorový profil

| Receptor | Ki (nM) | Účinek |
|----------|---------|--------|
| **5-HT2A** | 75-130 | Agonista (hlavní psychoaktivní) |
| **5-HT2C** | 360 | Agonista |
| **5-HT1A** | 1000+ | Slabý agonista |
| **Sigma-1** | 14,750 | Agonista |
| **TAAR1** | ~5000 | Agonista |

### Mechanismus účinku

```
DMT
  ↓
5-HT2A receptor (kortex)
  ↓
Gq protein → PLC → IP₃ + DAG
  ↓
Změna neuronální aktivity
  ↓
Halucinace, altered states
```

### Farmakokinetika

| Cesta podání | Nástup | Trvání | Biologická dostupnost |
|--------------|--------|--------|----------------------|
| **Inhalace** | 15-60 s | 5-20 min | ~100% |
| **Injekce (i.m.)** | 2-5 min | 30-45 min | ~100% |
| **Orálně + MAOI** | 30-60 min | 3-6 h | ~50% (s MAOI) |
| **Orálně (samotné)** | - | - | ~0% (MAO degradace) |

### MAO degradace

DMT je rapidně degradováno **monoaminooxidázou A** (MAO-A):

```
DMT → [MAO-A] → Indol-3-acetaldehyd → IAA
```

**Řešení**: Kombinace s MAO inhibitory (ayahuasca, harmalin)

## Strassmanovy studie (1990-1995)

### Design studie

Dr. Rick Strassman na Univerzitě v Novém Mexiku:

| Parametr | Hodnota |
|----------|---------|
| **Účastníci** | 60 dobrovolníků |
| **Dávky** | 0,05-0,4 mg/kg i.v. |
| **Administrace** | Intravenózní |
| **Měření** | Fyziologické + subjektivní |

### Klíčové nálezy

1. **Reprodukovatelnost** - Podobné zážitky u různých subjektů
2. **Dávková závislost** - Vyšší dávka = intenzivnější zážitek
3. **Entity** - 50% reportovalo setkání s bytostmi
4. **Bezpečnost** - Žádné závažné nežádoucí účinky

### Fyziologické změny

| Parametr | Změna |
|----------|-------|
| Krevní tlak | ↑ Zvýšení |
| Srdeční frekvence | ↑ Zvýšení |
| Tělesná teplota | ↑ Mírné zvýšení |
| Kortizol | ↑ Zvýšení |
| Prolaktin | ↑ Zvýšení |
| β-endorfin | ↑ Zvýšení |
