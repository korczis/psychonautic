# Ketamin - Disociativni anestetikum

**Ketamin** je disociativni anestetikum a antagonista **NMDA receptoru** s unikátnimi farmakologickymi vlastnostmi. Puvodne vyvinuty jako anestetikum (1962), dnes nachazi sirsi vyuziti v lecbe **rezistentni deprese**, chronicke bolesti a PTSD. Ketamin predstavuje jeden z nejvyznamnejsich prurvu v psychiatrii poslednich dekad.

## Fenomenologie

### Disociativni zkusenost

| Dávka | Uroven | Fenomenologie |
|-------|--------|---------------|
| **0,1-0,3 mg/kg** | Sub-hole | Relaxace, mírna euforie |
| **0,3-0,5 mg/kg** | Terapeuticka | Disociace, zmeny vnimani |
| **0,5-1 mg/kg** | Stredni | Silná disociace, vizuální změny |
| **1-2 mg/kg** | K-hole | Uplna disociace, NDE-like |
| **>2 mg/kg** | Anesteticka | Bezvedomi, anestezie |

### "K-hole" fenomen

Intenzivni disociativni zazitek pri vysich davkach:

| Charakteristika | Popis |
|-----------------|-------|
| **Ego disoluce** | Ztrata pocitu sebe sama |
| **Telesna disociace** | Odpojeni od fyzickeho tela |
| **Casova distorze** | Zamrzly cas, venost |
| **Vizualni** | Tunely, geometrie, "jine dimenze" |
| **NDE-like** | Podobne zazitku blizke smrti |
| **Kosmicke vedomi** | Pocit jednoty s vesmirem |
| **Amnézie** | Castecna retrogradui amnézie |
