# Aeruginascin - Kvarternalni tryptamin

**Aeruginascin** je unikatni **kvarternalni tryptaminovy alkaloid** objeveny v nekterych druzich rodu Psilocybe a Inocybe. Strukturne predstavuje **trimethylamoniovy analog** [psilocybinu](/alkaloids/psilocybin/) a pravdepodobne hraje roli v **entourage effectu** psychedelickych hub.

---

## Chemicka struktura

### Zakladni parametry

| Vlastnost | Hodnota |
|-----------|---------|
| **Chemicky nazev** | N,N,N-Trimethyl-4-fosforyloxytryptaminium |
| **Alternativni nazvy** | 4-Phosphoryloxy-N,N,N-trimethyltryptamine |
| **Molekularni vzorec** | C13H20N2O4P+ |
| **Molekularni hmotnost** | 299,28 g/mol |
| **Naboj** | +1 (kvarternalni amoniova sol) |
| **IUPAC** | [3-[2-(trimethylammonio)ethyl]-1H-indol-4-yl] phosphate |
| **CAS cislo** | 59299-44-4 |

### Strukturni srovnani s pribuznyni alkaloidy

| Sloucenina | N-substituenty | Naboj | Aktivita |
|------------|----------------|-------|----------|

| [Baeocystin](/alkaloids/baeocystin/) | NHCH3 | 0 | Slabsi nez psilocybin |

### Etymologie

Nazev "aeruginascin" je odvozen od:
- **Inocybe aeruginascens** - prvni znamy zdroj
- **"Aerugo"** (latinsky) = "med, zelenkava rez" (barva spor)
- **"-ascin"** = suffix pro alkaloidy z hub

### Inocybe aeruginascens

Hlavni prirodni zdroj aeruginascinu:

| Vlastnost | Hodnota |
|-----------|---------|
| **Celed** | Inocybaceae |
| **Habitat** | Pisecne pudy, parky, listnatne lesy |
| **Rozsirenosi** | Stredni Evropa (Nemecko, Polsko) |
| **Sezona** | Leto-podzim |
| **Alkaloidovy profil** | Psilocybin (0.5-1.2%), Aeruginascin (0.15-0.4%), Baeocystin |

**Historicka poznamka**: Jochen Gartz zaznamenal, ze *I. aeruginascens* vyvolaval prevazne **euforicke** zazitky bez "bad tripu", coz vedlo k hypoteze o protektivni roli aeruginascinu.

## Farmakologie

### Receptorova afinita

Aeruginascin sam o sobe pravdepodobne **nepusobi primo na 5-HT2A receptor** kvuli svemu kvaternarnimu naboji, ktery braneni prostupu hematoencefalickou barierou (BBB).

| Aspekt | Hodnota | Poznamka |
|--------|---------|----------|
| **5-HT2A afinita** | Nizka/zadna | Kvarternalni naboj |
| **Prostup BBB** | Minimalni | Hydrofilie |
| **Aktivita** | Modulacni? | Entourage effect |

### Potencialni metabolismus

```
Aeruginascin (N+-trimethyl)
        |
        v [Neznamy enzym? Spontanni?]
4-Fosforyloxy-N,N-dimethyltryptamin (PSILOCYBIN?)

        |

**Hypoteza**: Aeruginascin by mohl byt **prodrug** nebo **modulator** metabolismu psilocybinu.

```
