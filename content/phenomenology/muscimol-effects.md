# Účinky muscimolu - Fenomenologie GABAergního psychoaktiva

**Muscimol** představuje jedinečnou třídu psychoaktivních látek - **GABAergní psychoaktivum** - které se fundamentálně odlišuje od serotoninových psychedelik ([psilocybin](@/alkaloids/psilocybin.md), [LSD](@/alkaloids/lsd.md), [DMT](@/alkaloids/dmt.md)) svým mechanismem účinku i fenomenologickým profilem.

Zatímco klasická psychedelika aktivují [5-HT2A receptory](@/receptors/5-ht2a.md) a indukují vizuální halucinace s zachovanou orientací, muscimol jako agonista [GABA-A receptorů](@/receptors/gaba-a.md) vyvolává **oneirogenní** (snům podobné) stavy s výraznou sedací a alterací propriocepce.

---

## Základní farmakologický profil

### Mechanismus účinku

```
MUSCIMOL: GABAergní mechanismus

            GABA-A receptor
         +------------------+
         |     α β α β γ    |
         |      \|/|/       |
Muscimol -----> [ORTOSTERICKÉ VAZEBNÉ MÍSTO]
         |                  |
         |  ↓↓↓ Cl⁻ influx  |
         +------------------+
                 ↓
    HYPERPOLARIZACE NEURONU
                 ↓
    INHIBICE NEURONÁLNÍ AKTIVITY
                 ↓
┌─────────────────────────────────────────┐
│  KLINICKÉ EFEKTY DLE SUBTYPŮ           │
├─────────────────────────────────────────┤
│  α1 subtypy → SEDACE, amnézie          │
│  α2/α3 subtypy → ANXIOLÝZA             │
│  α4/α6 subtypy → Thalamická modulace   │
│  α5 subtypy → Kognitivní efekty        │
└─────────────────────────────────────────┘
```

| Parametr | Hodnota |
|----------|---------|
| **Primární cíl** | [GABA-A receptor](@/receptors/gaba-a.md) |
| **Typ vazby** | Ortosterický agonista |
| **Afinita (Ki)** | 2-10 nM |
| **Účinnost (Emax)** | ~80% vs GABA |
| **Onset** | 30-90 minut (p.o.) |
| **Trvání** | 4-8 hodin |
| **Peak efekt** | 2-3 hodiny |

### Srovnání s jinými GABAergiky

| Látka | Vazebné místo | Selektivita | Typ efektu |
|-------|---------------|-------------|------------|
| **Muscimol** | Ortosterické (GABA site) | Nízká | Plný agonismus |
| **Benzodiazepiny** | Alosterické (BZD site) | α1 preference | Pozitivní modulace |
| **Barbituráty** | Alosterické | Nízká | Přímá aktivace (vysoké dávky) |
| **Alkohol** | Alosterické + NMDA | Nízká | Pozitivní modulace |

## Typické účinky

### Pozitivní účinky

#### Hluboká relaxace

**Mechanismus**: Generalizovaná inhibice CNS přes GABA-A receptory

```
SPEKTRUM RELAXACE

Lehká relaxace ←──────────────────→ Hluboká sedace
     |                                    |
  α2/α3 anxiolýza              α1 sedace + amnézie
     |                                    |
  Zachovaná bdělost            Oneirogenní stavy
```

| Stupeň | Dávka (sušená Amanita) | Subjektivní popis |
|--------|------------------------|-------------------|
| **Lehká** | 1-3 g | Mírná relaxace, snížená úzkost |
| **Střední** | 3-5 g | Výrazná svalová relaxace, "těžké tělo" |
| **Silná** | 5-10 g | Hluboká sedace, obtížné udržet bdělost |

**Fenomenologický popis**:
> "Pocit absolutního uvolnění, jako by veškeré napětí v těle zmizelo. Tělo se cítí těžké a zároveň lehké - paradoxní stav, kdy gravitace působí silněji, ale zároveň nejsou žádné nepříjemné pocity."

#### Svalová relaxace

**Mechanismus**: GABA-A receptory v motorických neuronech míchy a mozkového kmene

| Oblast | Receptor subtyp | Efekt |
|--------|-----------------|-------|
| **Spinální mícha** | α2/α3 | Snížení svalového tonu |
| **Mozeček** | α6 | Koordinace (ataxie při vysokých dávkách) |
| **Bazální ganglia** | α1/α2 | Motorická inhibice |

**Terapeutický potenciál**:
- Svalové spasmy
- Spasticita (experimentální)
- Chronické napětí

#### Euforie

**Mechanismus**: Disinhibice reward okruhů, modulace dopaminergních drah

```
GABAergní EUFORIE vs PSYCHEDELICKÁ EUFORIE

MUSCIMOL                          PSILOCYBIN
    |                                  |
GABA-A inhibice              5-HT2A aktivace
    ↓                                  ↓
Disinhibice VTA              Kortikální excitace
    ↓                                  ↓
"Teplá" euforie              "Jasná" extáze
Sedativní                    Stimulující
Introspektivní               Expansivní
```

**Charakteristika muscimolové euforie**:
- "Teplý" pocit spokojenosti
- Absence úzkosti
- Introspektivní kvalita
- Kompatibilní se spánkem

#### Snové/oneirické stavy

**Definice**: Oneirogenní = vyvolávající stavy podobné snění při zachované (částečné) bdělosti

```
ONEIRICKÉ KONTINUUM

Plná bdělost → Hypnagogie → Oneirický stav → Lucidní sen → REM spánek
                    |              |               |
              Muscimol       Muscimol        Muscimol
              (nízké)        (střední)       (indukce)
```

| Fenomén | Popis | Frekvence |
|---------|-------|-----------|
| **Hypnagogické vize** | Živé obrazy při usínání | Velmi častá |
| **Oneirická halucinace** | Snové sekvence při bdělosti | Častá |
| **Lucidní prvky** | Vědomí snového stavu | Střední |
| **Konfabulace** | Míšení snu a reality | Častá |

**Rozdíl od psychedelických vizí**:

| Aspekt | Muscimol (oneirické) | 5-HT2A (psychedelické) |
|--------|---------------------|------------------------|
| **Charakter** | Snové, narativní | Geometrické, fraktální |
| **Orientace** | Snížená | Zachovaná |
| **Kontrola** | Minimální | Částečná |
| **Paměť** | Fragmentární | Zachovaná |

#### Spirituální zážitky

**Historický kontext**: Amanita muscaria v sibiřském a severském šamanismu

```
SPIRITUÁLNÍ DIMENZE MUSCIMOLU

┌─────────────────────────────────────────────┐
│  ŠAMANISTICKÉ VYUŽITÍ                       │
├─────────────────────────────────────────────┤
│  - Sibiřští shamani (Korjaci, Čukčové)      │
│  - Komunikace s duchy předků                │
│  - Divinace a věštění                       │
│  - Léčení (rituální kontext)                │
│  - Iniciační rituály                        │
└─────────────────────────────────────────────┘
```

**Fenomenologie spirituálních aspektů**:
- Pocit spojení s přírodou
- Animistické vjemy
- Komunikace s "duchy" (při vysokých dávkách)
- Archetypalní obrazy

#### Lucidní sny (následný efekt)

**Mechanismus**: Rebound REM spánku po GABAergní supresi

```
ČASOVÝ PRŮBĚH

Akutní fáze (0-6h)          Rebound fáze (6-24h)
        |                           |
  GABAergní sedace           Zvýšená REM aktivita
        ↓                           ↓
  Suprese REM               Intenzivní snění
        ↓                           ↓
  Snížená snová aktivita    LUCIDNÍ SNY
```

**Praktická doporučení pro indukci lucidních snů**:
1. Nízká až střední dávka (1-3 g sušené Amanity)
2. Užití 4-6 hodin před spaním
3. Wake-Back-To-Bed (WBTB) technika
4. Reality checking

| **Silná** | Obtížné udržet bdělost | Výrazný |

#### Změny vnímání velikosti (makro/mikropsie)

**Alice in Wonderland Syndrome (AIWS)**

```
ZMĚNY VNÍMÁNÍ VELIKOSTI

MAKROPSIE                         MIKROPSIE
    |                                  |
Objekty se zdají VĚTŠÍ      Objekty se zdají MENŠÍ
    |                                  |
"Ruce jsou obrovské"        "Pokoj je miniaturní"
    |                                  |
Proprioceptivní distorze    Vizuální distorze
```

**Mechanismus**: Thalamická modulace senzorické integrace přes α4/α6 GABA-A subtypy

| Typ | Frekvence | Popis |
|-----|-----------|-------|
| **Somatická makropsie** | Častá | Části těla se zdají větší |
| **Somatická mikropsie** | Střední | Části těla se zdají menší |
| **Vizuální makropsie** | Střední | Objekty se zdají větší |
| **Vizuální mikropsie** | Méně častá | Objekty se zdají menší |

**Literární paralela**: Lewis Carroll a "Alenka v říši divů" - Carroll pravděpodobně trpěl migrénami s AIWS, ale spekulace o inspiraci Amanitou nejsou podloženy.

#### Změny propriocepce

**Definice**: Propriocepce = vnímání polohy a pohybu vlastního těla

```
PROPRIOCEPTIVNÍ ALTERACE

Normální stav              Muscimolový stav
     |                           |
 Přesná lokalizace        Nejistá lokalizace
     |                           |
 Jasné hranice těla       Rozmlžené hranice
     |                           |
 Stabilní "body map"      Fluidní "body map"
```

**Fenomény**:
- Pocit "plovoucího" těla
- Nejistota polohy končetin
- Změny vnímání tělesných proporcí
- Disociace od těla (při vyšších dávkách)

#### Ztráta pojmu času

**Temporální distorze**:

| Typ | Popis | Mechanismus |
|-----|-------|-------------|
| **Dilatace** | Čas plyne pomaleji | Snížená kortikální aktivita |
| **Kontrakce** | Čas plyne rychleji | Amnestické "mezery" |
| **Fragmentace** | Diskontinuální vnímání | Oneirické epizody |

### Negativní účinky

#### Ataxie (porucha koordinace)

**Mechanismus**: Cerebelární inhibice přes α6-obsahující GABA-A receptory

```
STUPNĚ ATAXIE

Stupeň 0    Stupeň 1       Stupeň 2       Stupeň 3
    |           |              |              |
 Normální   Mírná          Výrazná        Těžká
    |           |              |              |
Stabilní    Nejistá        Potřeba        Neschopnost
chůze       chůze          opory          stát
```

**Rizika**:
- Pády a zranění
- Aspirace (při zvracení)
- Hypotermie (neschopnost pohybu)

**Harm reduction**:
- Sedět nebo ležet
- Bezpečné prostředí bez schodů
- Trip sitter

#### Nauzea a zvracení

**Příčiny**:
1. **Kyselina ibotenová** - neurotoxický prekurzor (více při nedostatečné dekarboxylaci)
2. **Gastrointestinální dráždění** - přímý efekt
3. **Vestibulární dysregulace** - závratě → nauzea

```
MINIMALIZACE NAUZEY

Špatná příprava              Správná příprava
      |                            |
  Syrová Amanita            Sušená/vařená
      |                            |
  Vysoká kyselina           Nízká kyselina
  ibotenová                 ibotenová
      |                            |
  SILNÁ NAUZEA              MÍRNÁ/ŽÁDNÁ NAUZEA
```

**Doporučení**:
- Důkladné sušení (70°C, minimálně 24h)
- Možnost extrakce do vody (kyselina ibotenová je rozpustná)
- Antiemetika (zázvor, ondansetron)

#### Závratě

**Mechanismus**: Vestibulární inhibice + ortostatická hypotenze

| Typ | Popis | Management |
|-----|-------|------------|
| **Vestibulární** | Točení hlavy | Ležet, zavřít oči |
| **Ortostatická** | Při vstávání | Pomalu měnit polohu |
| **Cerebelární** | Nejistota v prostoru | Stabilní prostředí |

#### Dezorientace

**Spektrum dezorientace**:

```
KONTINUUM DEZORIENTACE

Orientovaný → Mírná nejistota → Konfuze → Delirium
     |              |               |          |
  Normální    "Kde jsem?"      Neschopnost   Halucinace
  kognice     ale rychle       rozlišit      + dezorientace
              vzpamatování     sen/realitu
```

| Doména | Lehká | Střední | Těžká |
|--------|-------|---------|-------|
| **Místo** | ✓ | ± | ✗ |
| **Čas** | ± | ✗ | ✗ |
| **Osoba** | ✓ | ✓ | ± |
| **Situace** | ✓ | ± | ✗ |

#### Svalové záškuby

**Myoklonus**: Nedobrovolné svalové kontrakce

**Mechanismus**: Paradoxní excitace při neúplné GABAergní inhibici nebo rebound aktivita

| Typ | Popis | Čas výskytu |
|-----|-------|-------------|
| **Hypnický** | Při usínání | Onset, offset |
| **Generalizovaný** | Celé tělo | Peak efekt |
| **Fokální** | Jedna oblast | Variabilní |

#### Amnézie

**Typy amnézie**:

```
AMNÉZIE U MUSCIMOLU

ANTEROGRÁDNÍ                    RETROGRÁDNÍ
     |                               |
Neschopnost tvořit          Ztráta starších
nové vzpomínky              vzpomínek (vzácná)
     |                               |
Typická pro muscimol        Méně častá
     |
 "Blackout" epizody
```

**Mechanismus**: α1 a α5 subtypy GABA-A v [hippocampu](@/brain/hippocampus.md)

**Klinický význam**:
- Podobné jako u benzodiazepinů
- Dose-dependent
- Fragmentární vzpomínky vs úplný blackout

## Harm reduction

### Dávkování

| Stupeň | Sušená Amanita | Čistý muscimol |
|--------|----------------|----------------|
| **Prahová** | 0.5-1 g | 3-5 mg |
| **Lehká** | 1-3 g | 5-10 mg |
| **Střední** | 3-5 g | 10-15 mg |
| **Silná** | 5-10 g | 15-25 mg |
| **Velmi silná** | >10 g | >25 mg |

### Bezpečnostní opatření

1. **Příprava**
   - Důkladné sušení (dekarboxylace kyseliny ibotenové)
   - Identifikace druhu (pozor na jedovaté záměny)

2. **Set & setting**
   - Bezpečné, tiché prostředí
   - Trip sitter
   - Bez schodů, ostrých předmětů

3. **Interakce**
   - **Kontraindikace**: Alkohol, benzodiazepiny, opioidy
   - **Opatrnost**: Antidepresiva, antipsychotika

4. **Zdravotní stav**
   - Neužívat při respiračních onemocněních
   - Opatrnost při epilepsii

| Oblast | Status | Mechanismus |
