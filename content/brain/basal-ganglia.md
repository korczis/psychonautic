# Bazální ganglia - GABAergní motorické jádra

**Bazální ganglia** (BG) jsou skupina subkortikálních jader klíčových pro **motorickou kontrolu**, **procedurální učení**, **návyky** a **kognitivní funkce**.
Jedná se o jeden z nejvýznamnějších **GABAergních systémů** mozku - striatum obsahuje téměř výhradně GABAergní projekční neurony (medium spiny neurons, MSN).

Dysfunkce BG vede k motorickým poruchám (Parkinsonova choroba, Huntingtonova choroba, dystonie).

## GABAergní organizace

### Medium Spiny Neurons (MSN)

```
STRIATÁLNÍ MEDIUM SPINY NEURONS

MSN POPULACE:
═════════════
~95% striatálních neuronů = MSN
~50% D1-MSN (přímá dráha)
~50% D2-MSN (nepřímá dráha)

┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   D1-MSN (Striatonigral)          D2-MSN (Striatopallidal) │
│   ════════════════════            ══════════════════════   │
│                                                             │
│   Receptor: D1 (Gs)               Receptor: D2 (Gi)        │
│   Ko-transmiter: SP, Dyn          Ko-transmiter: Enk       │
│                                                             │
│   Projekce: GPi/SNr (přímá)       Projekce: GPe (nepřímá)  │
│                                                             │
│   Dopamin efekt:                  Dopamin efekt:           │
│   DA → D1 → ↑ cAMP → ↑ MSN       DA → D2 → ↓ cAMP → ↓ MSN │
│   = Facilitace pohybu            = Inhibice pohybu         │
│                                                             │
│        ┌─────────┐                     ┌─────────┐         │
│        │   D1    │                     │   D2    │         │
│        │   MSN   │                     │   MSN   │         │
│        └────┬────┘                     └────┬────┘         │
│             │                               │              │
│         GABA + SP                       GABA + Enk         │
│             │                               │              │
│             ▼                               ▼              │
│        ┌─────────┐                     ┌─────────┐         │
│        │GPi/SNr  │                     │   GPe   │         │
│        │(výstup) │                     │ (relay) │         │
│        └─────────┘                     └─────────┘         │
│                                                             │
└─────────────────────────────────────────────────────────────┘

GABA-A RECEPTORY NA MSN:
═══════════════════════
• α1, α2: Střední exprese
• α4: Vysoká (charakteristické pro striatum)
• α5: Vysoká (charakteristické pro striatum)
• β2, β3: Vysoká
• γ2: Vysoká
• δ: Střední (tonická inhibice)
```

### Muscimol a bazální ganglia

```

MECHANISMUS:

Přímá GABA-A aktivace ve všech strukturách BG

│                                                            │

│  ═════════════════════════                                │
│                                                            │

│     ████████████████████████████                          │
│     → ↑↑ Inhibice výstupních neuronů                     │
│     → ↓ GABA do thalamu                                   │
│     → Thalamická disinhibice (!)                         │
│                                                            │
│  4. STN                                                   │
│     ████████████████████████                              │
│     → ↑ Inhibice (od GPe)                                │
│     → ↓ Glu do GPi/SNr                                    │
│                                                            │
│  5. SNc (DA neurony)                                      │
│     ████████████████████                                  │

│     → ↓ DA release ve striatu                            │
│                                                            │
└────────────────────────────────────────────────────────────┘

NET EFEKT MUSCIMOLU NA BG:
═════════════════════════
Komplexní, závisí na dávce a lokalitě:

Nízká dávka (systémová):
├── Mírná ataxie
├── ↓ Motorická iniciace
└── Svalová relaxace

Střední dávka:
├── Výrazná ataxie
├── Dyskoordinace
├── Pomalé pohyby (bradykineze-like)
└── Potíže se zahájením pohybu

Vysoká dávka:
├── Těžká motorická dysfunkce
├── Akineze
├── Katalepsie-like stavy
└── Úplná ztráta motorické kontroly
```

### Porovnání látek působících na BG

| Látka | Primární cíl | Efekt na DA | Efekt na GABA | Motorický efekt |
|-------|--------------|-------------|---------------|-----------------|
| **Muscimol** | GABA-A (globální) | ↓ (nepřímo) | ↑↑↑ | Ataxie, bradykineze |
| **L-DOPA** | → DA | ↑↑ | - | ↑ Pohyb, dyskineze |
| **Amfetamin** | DAT, VMAT | ↑↑↑ | - | Stereotypie, hyperkineze |
| **Haloperidol** | D2 antag. | - (blokáda) | - | Parkinsonismus |
| **Kokain** | DAT | ↑↑ | - | Stereotypie |
| **Alkohol** | GABA-A PAM | ↑ (akutně) | ↑ | Ataxie |
| **Benzodiazepiny** | GABA-A PAM | - | ↑ | Mírná ataxie |

## Patologie bazálních ganglií

### Parkinsonova choroba

```
PARKINSONOVA CHOROBA - PATOFYZIOLOGIE

PATOLOGIE:
══════════
Degenerace dopaminergních neuronů v SNc

                    NORMÁLNÍ               PARKINSON
                    ════════               ═════════

SNc:                █████████              ░░░░░░░░░
                    (DA neurony)           (ztráta >60%)

Striatum:
D1-MSN aktivita:    ████████               ████
D2-MSN aktivita:    ████████               ████████████

Přímá dráha:        AKTIVNÍ                HYPOAKTIVNÍ
Nepřímá dráha:      NORMÁLNÍ               HYPERAKTIVNÍ

GPi/SNr výstup:     ████████               ████████████████
                    (normální)             (↑↑ inhibice thalamu)

Thalamus:           AKTIVNÍ                HYPOAKTIVNÍ

Kortex:             Normální pohyb         HYPOKINEZE

MOTORICKÉ SYMPTOMY:
══════════════════
• Bradykineze (pomalé pohyby)
• Rigidita (svalová ztuhlost)
• Klidový tremor (4-6 Hz)
• Posturální instabilita
```

### Huntingtonova choroba

```
HUNTINGTONOVA CHOROBA - PATOFYZIOLOGIE

PATOLOGIE:
══════════
Degenerace MSN ve striatu (CAG repeat v HTT genu)

ČASNÁ FÁZE:
══════════
D2-MSN (nepřímá dráha) degenerují PRVNÍ

Striatum:
D1-MSN:             ████████████
D2-MSN:             ░░░░░░░░░░░░ (ztráta)

Nepřímá dráha:      HYPOAKTIVNÍ
Přímá dráha:        RELATIVNĚ ZACHOVÁNA

GPi/SNr výstup:     ████████████
                    ↓↓ (méně inhibice z nepřímé)

Thalamus:           HYPERAKTIVNÍ (disinhibice)

= CHOREA (nechtěné pohyby)

POZDNÍ FÁZE:
═══════════
Degenerace obou populací MSN

= RIGIDITA, BRADYKINEZE (jako Parkinson)
```
