# Retikulární formace - Strážce vědomí

**Retikulární formace** (RF) je difuzní síť neuronů táhnoucí se celým mozkovým kmenem od prodloužené míchy po mezimozek.
Její klíčovou součástí je **ascendentní retikulární aktivační systém (ARAS)**, který je fundamentální pro udržení **vědomí** a **bdělosti**.

GABAergní inhibice retikulární formace představuje jeden z hlavních mechanismů **anestezie**, účinku **muscimolu** a fyziologického **spánku**.

## Ascendentní retikulární aktivační systém (ARAS)

### Definice a historie

**ARAS** byl objeven v roce 1949 Moruzzim a Magounem.
Představuje systém ascendentních projekcí z mozkového kmene, které aktivují forebrain a udržují stav bdělosti.

### ARAS dráhy

| Dráha | Zdroj | Cíl | NT | Efekt na vědomí |
|-------|-------|-----|----|--------------------|
| **Dorzální** | PPTg/LDTg | Thalamus (IL) | ACh | ↑ Thalamická aktivace |
| **Ventrální** | LC, VTA, Raphe | BF, Kortex | NE, DA, 5-HT | ↑ Kortikální arousal |
| **Glutamátergní** | Parabrachialis | BF, Thalamus | Glu | ↑ Rychlá aktivace |
| **Histaminergní** | TMN | Kortex, Thalamus | Histamin | ↑ Bdělost |
| **Orexinergní** | LH | Široká | Orexin | ↑ Stabilizace bdělosti |

## GABAergní regulace ARAS

### GABA v retikulární formaci

```
GABAERGNÍ SYSTÉMY V RETIKULÁRNÍ FORMACI

┌────────────────────────────────────────────────────────────┐
│                                                            │
│  ZDROJE GABA INHIBICE NA ARAS:                            │
│  ═══════════════════════════════                          │
│                                                            │
│  1. VENTROLATERÁLNÍ PREOPTICKÁ OBLAST (VLPO)             │
│     ┌───────────────────────────────────────────┐         │
│     │  "Spánkový přepínač"                      │         │
│     │                                           │         │
│     │  VLPO ──GABA+Galanin──► ARAS jádra       │         │
│     │                         (LC, TMN, Raphe)  │         │
│     │                                           │         │
│     │  Aktivní během spánku                    │         │
│     │  → Inhibice arousal systémů              │         │
│     │  → Umožnění spánku                       │         │
│     └───────────────────────────────────────────┘         │
│                                                            │
│  2. PARAFACIÁLNÍ ZÓNA (PZ)                               │
│     ┌───────────────────────────────────────────┐         │
│     │  PZ ──GABA──► Parabrachialis             │         │
│     │                                           │         │
│     │  Nový spánkový centrum                   │         │
│     │  CO₂-senzitivní                          │         │
│     └───────────────────────────────────────────┘         │
│                                                            │
│  3. LOKÁLNÍ GABA INTERNEURONY                            │
│     ┌───────────────────────────────────────────┐         │
│     │  V každém ARAS jádře:                    │         │
│     │  • LC: GABA interneurony                 │         │
│     │  • PPTg: GABA neurony                    │         │
│     │  • Raphe: GABA interneurony              │         │
│     │                                           │         │
│     │  Lokální regulace, gain control          │         │
│     └───────────────────────────────────────────┘         │
│                                                            │
└────────────────────────────────────────────────────────────┘

SPÁNKOVÝ "FLIP-FLOP" PŘEPÍNAČ:
══════════════════════════════

    BDĚLOST                           SPÁNEK
    ────────                          ──────

    ARAS jádra                        VLPO
    (LC, TMN, Raphe)                  (GABA + Galanin)
         │                                 │
         │  ◄──── Vzájemná inhibice ────►  │
         │                                 │
    ┌────┴────┐                       ┌────┴────┐
    │  AKTIVNÍ │                       │  AKTIVNÍ │
    │  →Bdělost│                       │  →Spánek │
    └─────────┘                       └─────────┘

Orexin: Stabilizuje "bdělou" pozici přepínače
(deficit orexinu = narkolepsie)
```

### GABA-A receptory v ARAS jádrech

| Jádro | α1 | α2 | α3 | β2/3 | γ2 | Funkce GABA |
|-------|----|----|-------|------|----|--------------------|
| **LC** | ++ | + | ++ | +++ | ++ | ↓ NE release, sedace |
| **PPTg/LDTg** | +++ | + | ++ | +++ | ++ | ↓ ACh, ↓ REM |
| **TMN** | ++ | + | + | ++ | ++ | ↓ histamin, sedace |
| **Raphe dorsalis** | ++ | ++ | + | ++ | ++ | ↓ 5-HT release |
| **VTA** | ++ | + | ++ | +++ | ++ | ↓ DA release |
| **Parabrachialis** | +++ | + | +++ | +++ | ++ | Klíčový pro anestetika |
