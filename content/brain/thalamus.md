# Thalamus - Brana vedomi

**Thalamus** (z reckeho "thalamos" - komora) je parova struktura mezimozku, ktera slouzi jako hlavni **relay stanice** pro senzorické informace smerujici do kortexu. Krome senzorickeho zpracovani hraje klicovou roli v **regulaci vedomi**, **pozornosti**, **spanku** a **kortikalni synchronizaci**.

V kontextu psychedelik je thalamus dulezity jako "brana" filtrovani informaci - jeho modulace muze vest ke zmenam v tom, co a jak vnimame.

## Jadra thalamu

### Hlavni jadrove skupiny

| Skupina | Jadra | Hlavni funkce |
|---------|-------|---------------|
| **Anteriorni** | AN | Emocni pamet, Papezův okruh |
| **Medialni** | MD | Exekutivni funkce, PFC |
| **Laterální dorzální** | LD, LP | Parietalni asociace |
| **Ventrolaterální** | VA, VL | Motoricka kontrola |
| **Ventrobazální** | VPL, VPM | Somatosenzorický relay |
| **Posteriorni** | Pulvinar, LGN, MGN | Vizualni, sluchovy relay |
| **Intralaminární** | CM, Pf, CL | Arousal, vedomi |
| **Retikulární (TRN)** | TRN | Inhibice, gate |

## Senzoricke drahy

### Thalamokortikalni relay

```
SENZORICKY RELAY SKRZE THALAMUS

Vizualni:     Retina → LGN → V1 (occipitalni)

Sluchovy:     Cochlea → MGN → A1 (temporalni)

Somatosenz.:  Telo → VPL → S1 (parietalni)
Tvár:         Tvár → VPM → S1

Cich:         (VYJIMKA - primo do kortexu)
              Nos → Olfactory bulb → Piriform cortex

Bolest:       Nocicept. → VPL/VPM + IL jadra
```

### Lateralni genikulatni jadro (LGN)

| Vrstva | Typ | Funkce |
|--------|-----|--------|
| **1-2** | Magnocelulární | Pohyb, kontrast |
| **3-6** | Parvocelulární | Barva, detail |
| **Konicelulární** | Mezi vrstvami | Barva (S-cone) |

### Specificka jadra detail

| Jadro | Vstup | Vystup | Funkce |
|-------|-------|--------|--------|
| **LGN** | Retina (opticky nerv) | V1 | Vizualni relay |
| **MGN** | Inferior colliculus | A1 | Sluchovy relay |
| **VPL** | Spinalni draha | S1 | Telesna senzace |
| **VPM** | Trigeminalni | S1 | Tvar, usta |
| **VL** | Cerebellum, GPi | M1 | Motoricky plan |
| **MD** | [Amygdala](/brain/amygdala/), PFC | [PFC](/brain/prefrontal-cortex/) | Exekutivni |

## Thalamicky gate

### Retikulární jadro (TRN)

**Thalamicke retikulární jadro** je klicove pro "gating" - filtrovani informaci:

| Vlastnost | Hodnota |
|-----------|---------|
| **Pozice** | Obaluje thalamus zvenci |
| **Neurotransmiter** | GABA (inhibicni) |
| **Vstup** | Kortex, thalamus |
| **Vystup** | Pouze do thalamu (inhibice) |
| **Funkce** | Selektivni pozornost, gate |

```
THALAMICKY GATE MODEL

Kortex
   │
   │ Kortikothalamický
   ↓
┌──────────────────────────────────────────┐
│                  TRN                      │
│           (GABAergni shell)              │
│                   │                       │
│          Inhibice │                       │
│                   ↓                       │
│   ┌─────────────────────────────────┐    │
│   │        THALAMICKÁ JADRA         │    │
│   │                                 │    │
│   │   Senzorický   →   Kortikální  │    │
│   │     vstup          projekce    │    │
│   └─────────────────────────────────┘    │
└──────────────────────────────────────────┘

TRN = "Gate keeper"
- Rozhoduje, co projde do kortexu
- Ovládán pozorností, kortexem
- Modulován psychedeliky
```

### Attention gating

| Stav | TRN aktivita | Thalamicky vystup |
|------|--------------|-------------------|
| **Fokusovaná pozornost** | Selektivni inhibice | Filtrovaný, relevantni |
| **Rozptylená pozornost** | Nizka | Nefiltrovaný |
| **Spanek (NREM)** | Vysoka oscilacni | Blokovany (odpojeni) |
| **Psychedelicky stav** | Modulovana | Zvyseny, nefiltrovaný |

## Thalamus a vedomi

### Integracni teorie

Thalamus hraje klicovou roli v teoriich vedomi:

| Teorie | Role thalamu |
|--------|--------------|
| **Global Workspace** | Integrace a broadcasting |
| **IIT** | Informacni integrace |
| **Dynamic Core** | Thalamokortikalni loop |
| **Thalamic Switch** | Gate vedomi |

### Thalamokortikalni oscilace

```
STAV VEDOMI A OSCILACE

Bdely stav (vedomi):
Thalamus ←──── 40 Hz (gamma) ────→ Kortex
         │                        │
         └────── Rezonance ───────┘

Anestezie/koma (bezvédomi):
Thalamus ←──── <10 Hz ─────→ Kortex
         │                   │
         └── Desynchronizace ┘

Psychedelicky stav:
Thalamus ←──── Zvysena entropie ──→ Kortex
         │                         │
         └── Narusena filtrace ────┘
```

### Thalamicky gate a anestezie

| Anestetikum | Mechanismus | Thalamicky efekt |
|-------------|-------------|------------------|
| **Propofol** | GABA-A PAM | Hyperpolarizace, burst |
| **Sevofluran** | Multiple | Thalamokort. odpojeni |
| **Ketamin** | [NMDA](/receptors/nmda/) antagonista | Disociace |

## Thalamus a psychedelika

### Efekty psychedelik

Psychedelika ([psilocybin](/alkaloids/psilocybin/), [LSD](/alkaloids/lsd/)) meni thalamickou funkci:

| Efekt | Mechanismus | Dusledek |
|-------|-------------|----------|
| **Zvysena konektivita** | 5-HT2A aktivace | Nefiltrovaný senz. vstup |
| **Oslabeny gating** | TRN modulace | "Otevřena brána" |
| **Zmenene oscilace** | Desynchronizace | Zmenene vedomi |
| **Hyperkonektivita** | Glob. zmeny | Synestezie |

### Neurozobrazovaci nalezy

```
THALAMICKA KONEKTIVITA POD PSYCHEDELIKY

Normalni stav:
┌─────────────────────────────────────────┐
│ Kortex ←── Selektivni ──→ Thalamus     │
│            (filtrovaná)                 │
│                                         │
│ Modalita A │         │ Modalita B      │
│     ↓      │         │      ↓          │
│  Oblast A  │  TRN    │   Oblast B      │
│            │ (gate)  │                  │
└─────────────────────────────────────────┘

Pod psilocybinem:
┌─────────────────────────────────────────┐
│ Kortex ←── Zvysena ──→ Thalamus        │
│            (nefiltrovaná)               │
│        ╲             ╱                  │
│         ╲           ╱                   │
│          ╲         ╱                    │
│           ╲       ╱                     │
│ Oblast A ←──┴───┴──→ Oblast B          │
│                                         │
│ Hyperkonektivita, "cross-talk"          │
└─────────────────────────────────────────┘
```

## Patologie thalamu

### Thalamicky infarkt

| Arterie | Postizena jadra | Symptomy |
|---------|-----------------|----------|
| **Tuberothalamicka** | Anteriorni, MD | Amnezie, apatie |
| **Paramedianní** | MD, IL | Somnolence, poruchy pameti |
| **Inferolateralni** | VPL, VPM | Senzoricky deficit |
| **Posteriorni** | Pulvinar, LGN | Vizualni deficity |

### Thalamická bolest (Dejerine-Roussy)

Po lezi VPL/VPM:
- Silna, palivá bolest
- Kontralaterální
- Centralni post-stroke bolest
- Obtizne lecitelná

### Poruchy vedomi

| Leze | Dusledek |
|------|----------|
| **Bilateralni IL jadra** | Koma, vegetativni stav |
| **Unilateralni thalamus** | Neglect, attention deficity |
| **Pulvinar** | Vizuospatial deficity |

### Fatal Familial Insomnia

Prionove onemocneni s predilekcí pro thalamus:
- Progresivni insomnie
- Autonomni dysfunkce
- Demence
- Smrt
