# Set and Setting - Zaklad psychedelicke zkusenosti

**Set and Setting** je fundamentalni koncept v psychedelicke vede a terapii, ktery popisuje dva klicove faktory ovlivnujici charakter a vysledek psychedelicke zkusenosti: **set** (vnitrni nasteveni, mindset) a **setting** (vnejsi prostredi). Tento koncept, popularizovany Timothy Learym v 60. letech, se stal zakladnim kamenem modernich klinickych protokolu a harm reduction praktik.

Pochopeni a optimalizace set and setting je nezbytne pro maximalizaci terapeutickeho potencialu [psychedelik](/alkaloids/) a minimalizaci rizika nezadoucich ucinku.

---

## Historicky kontext

### Timothy Leary a Harvard Psilocybin Project

Koncept set and setting formuloval **Timothy Leary** (1920-1996) behem Harvard Psilocybin Project (1960-1962), kde studoval ucinky [psilocybinu](/alkaloids/psilocybin/) na lidske vedomi.

| Datum | Udalost | Vyznam |
|-------|---------|--------|
| **1960** | Zahajeni Harvard Psilocybin Project | Systematicky vyzkum psilocybinu |
| **1961** | Good Friday Experiment (Walter Pahnke) | Demonstrace vlivu settingu |
| **1963** | Vyhozeni z Harvardu | Kontroverzni metodologie |
| **1964** | Publikace "The Psychedelic Experience" | Formalizace konceptu |

### Puvodní formulace

Leary definoval set and setting takto:

> "The nature of the [psychedelic] experience depends almost entirely on set and setting. Set denotes the preparation of the individual, including his personality structure and his mood at the time. Setting is physical - the weather, the room's atmosphere; social - feelings of persons present towards one another; and cultural - prevailing views as to what is real."

### Evoluce konceptu

| Obdobi | Pojetí | Klicove prispevky |
|--------|--------|-------------------|
| **1960s** | Kontrakulturnní | Leary, Alpert, Metzner |
| **1970s** | Underground harm reduction | MAPS, Zendo Project |
| **1990s** | Renesance vyzkumu | Griffiths, Nichols, Vollenweider |
| **2010s** | Standardizovane protokoly | COMPASS, Johns Hopkins |
| **2020s** | Evidence-based praxe | FDA Breakthrough Therapy |

## Set - Vnitrni nastaveni

### Definice

**Set** (z anglickeho "mindset") oznacuje vnitrni psychologicky stav jedince vstupujiciho do psychedelicke zkusenosti. Zahrnuje:

```
SET (Mindset) - Vnitrni faktory
    |
    +-- AKUTNI STAV
    |       |-- Aktualni nalada
    |       |-- Uzkost vs klid
    |       |-- Ocekavani
    |       +-- Zamer/intence
    |
    +-- STABILNI CHARAKTERISTIKY
    |       |-- Osobnostni rysy
    |       |-- Psychiatricka anamneza
    |       |-- Zkusenosti s psychedeliky
    |       +-- Copingove strategie
    |
    +-- PRIPRAVA
            |-- Psychoedukace
            |-- Terapeuticky vztah
            |-- Intencni prace
            +-- Telesna priprava
```

### Komponenty setu

#### 1. Akutni psychologicky stav

| Faktor | Pozitivni indikatory | Rizikove indikatory |
|--------|---------------------|---------------------|
| **Nalada** | Stabilni, neutralni az pozitivni | Depresivni, manic, iritabilni |
| **Uzkost** | Minimalni, zvladatelna | Vysoka, panicka |
| **Unavá** | Odpocaty, energicky | Vycerpany, spankovy deficit |
| **Motivace** | Jasna, intrinsicka | Externi tlak, unik |

#### 2. Ocekavani a intence

Ocekavani zasadne moduluji psychedelickou zkusenost:

| Typ ocekavani | Vliv na zkusenost | Priklad |
|---------------|-------------------|---------|
| **Pozitivni** | Zvysuje pravdepodobnost pozitivni zkusenosti | "Ocekavam hluboke uvědomeni" |
| **Negativni** | Zvysuje riziko negativni zkusenosti | "Bojim se ztraty kontroly" |
| **Neutralni** | Otevrenost k jakemukoli vysledku | "Jsem zvedavy, co prijde" |
| **Specificke** | Muze vest k frustaci, kdyz se nenaplni | "Chci videt boha" |

**Doporuceny pristup**: Kultivovat "bezmenne ocekavani" - otevrenost bez fixace na specificky vysledek.

#### 3. Intence (Zamer)

Jasna intence je klicovym prediktorem terapeutickeho vysledku:

```
FORMULACE INTENCE
    |
    +-- NECO, CO CHCI PROZKOUMAT
    |       "Proc jsem uvizly v tomto vztahu?"
    |
    +-- NECO, CO CHCI UVOLNIT
    |       "Chci se osvobodit od strachu ze smrti"
    |
    +-- NECO, CO CHCI ZISKAT
    |       "Chci najit hlubsi smysl zivota"
    |
    +-- NECO, CO CHCI PRIJMOUT
            "Chci se smirit s minulosti"
```

#### 4. Osobnostni faktory

| Rys (Big Five) | Vliv na psychedelickou zkusenost |
|----------------|----------------------------------|
| **Openness** | Vyssi = vetsinou pozitivnejsi zkusenosti |
| **Neuroticism** | Vyssi = zvysene riziko obtiznych zkusenosti |
| **Extraversion** | Smisene ucinky |
| **Agreeableness** | Vyssi = lepsi terapeuticky vztah |
| **Conscientiousness** | Vyssi = lepsi priprava a integrace |

#### 5. Psychiatricka anamneza

| Stav | Doporuceni | Zduvodneni |
|------|------------|------------|
| **Schizofrenie** | ABSOLUTNI KONTRAINDIKACE | Riziko psychoticke dekompenzace |
| **Bipolární porucha** | Relativni kontraindikace | Riziko manie |
| **Deprese** | Mozna indikace (klinicky) | Terapeutický potencial |
| **PTSD** | Mozna indikace (klinicky) | Terapeuticky potencial |
| **Uzkostne poruchy** | Opatrnost, nizsi davky | Zvysene riziko paniky |
| **Osobnostni poruchy** | Individualni posouzeni | Variabilni vysledky |

## Setting - Vnejsi prostredi

### Definice

**Setting** oznacuje veskere vnejsi faktory ovlivnujici psychedelickou zkusenost, vcetne fyzickeho, socialniho a kulturniho prostredi.

```
SETTING - Vnejsi faktory
    |
    +-- FYZICKE PROSTREDI
    |       |-- Mistnost/prostor
    |       |-- Osvetleni
    |       |-- Teplota
    |       |-- Zvuky/hudba
    |       +-- Bezpecnost
    |
    +-- SOCIALNI PROSTREDI
    |       |-- Pritomne osoby
    |       |-- Terapeut/pruvodce
    |       |-- Skupinova dynamika
    |       +-- Duvernost
    |
    +-- KULTURNI KONTEXT
            |-- Legalni status
            |-- Spolecenske vnimani
            |-- Ritualni ramec
            +-- Vyznamovy system
```

### Fyzicke prostredi

#### 1. Prostor

| Prvek | Optimalni nastaveni | Zduvodneni |
|-------|---------------------|------------|
| **Velikost** | Stredne velka mistnost (15-25 m2) | Ani stisnjujici, ani prilis otevrena |
| **Pohodli** | Pohodlne lezeni/sezeni, deky, polstare | Fyzicky komfort |
| **Soukromi** | Uzavreny prostor bez vnejsiho ruseni | Bezpecnost, intimita |
| **Bezpecnost** | Zadne ostre predmety, zajieten pristup k WC | Prevence urazu |

#### 2. Osvetleni

| Typ | Pouziti | Efekt |
|-----|---------|-------|
| **Tlumene** | Vetsina sezeni | Internalizace, relaxace |
| **Prirodni** | Denni sezeni, integrace | Uzemneni, orientace |
| **Svicky** | Ritualni kontext | Posvátná atmosfera |
| **Tma** | Hluboká introspekce | Maximalni interiorita |

#### 3. Zvukove prostredi

Hudba je mocnym nastrojem modulace zkusenosti:

| Faze zkusenosti | Typ hudby | Priklad |
|-----------------|-----------|---------|
| **Nastup (0-60 min)** | Jemna, ambientni | Brian Eno, Sigur Ros |
| **Peak (60-180 min)** | Komplexni, emocialni | Klasika, world music |
| **Sestup (180-300 min)** | Mekcí, integrujici | Akusticka, prirodni zvuky |
| **Navrat** | Ticho nebo minimalni | Prostor pro reflexi |

**Johns Hopkins Playlist**: Standardizovana 7hodinova playlistapoznavacena pro klinicke studie, volne dostupna.

#### 4. Teplota a komfort

| Faktor | Doporuceni |
|--------|------------|
| **Teplota** | 20-22°C (psychedelika casto snizuji telesnou teplotu) |
| **Deky** | K dispozici (fyzicke teplo = psychologicka bezpecnost) |
| **Voda** | Snadno dostupna |
| **Lehke obcerstveni** | Ovoce, orechy po sezeni |

### Socialni prostredi

#### 1. Pritomnost pruvodce/terapeuta

| Aspekt | Vyznam |
|--------|--------|
| **Bezpecnost** | Fyzicka a psychologicka ochrana |
| **Kotveni** | Pripominani reality, orientace |
| **Podpora** | Emocionalni validace, neverbalni uklidneni |
| **Svedek** | Potvrzeni zkusenosti, nehodnoticí pritomnost |

#### 2. Terapeuticky vztah

Kvalita vztahu mezi klientem a pruvodcem je klicovym prediktorem vysledku:

```
TERAPEUTICKY VZTAH
    |
    +-- DUVERA
    |       Klient se citi bezpecne
    |       Otevrenost k zranitelnosti
    |
    +-- RESPEKT
    |       Nehodnoticí pristup
    |       Autonomie klienta
    |
    +-- KOMPETENCE
    |       Zkusenost terapeuta
    |       Znalost psychedelik
    |
    +-- PRITOMNOST
            Plna pozornost
            Empaticke ladeni
```

#### 3. Skupinova vs individualni zkusenost

| Kontext | Vyhody | Nevyhody |
|---------|--------|----------|
| **Individualni** | Maximalni pozornost, intimita | Vyssi naklady |
| **Dyadicka (2 ucastnici)** | Sdilena zkusenost, nizsi naklady | Potencialni interference |
| **Skupinova (3-8)** | Komunita, sdileni | Snizena individualni pozornost |

### Kulturni kontext

#### 1. Ritualni vs klinicky ramec

| Aspekt | Tradiční rituál | Klinicky protocol |
|--------|-----------------|-------------------|
| **Prostredi** | Priroda, obradni prostor | Klinicka mistnost |
| **Pruvodce** | Saman, curandero | Psychoterapeut, psychiatr |
| **Ramec** | Spiritualni, kosmologicky | Medicinskky, psychologicky |
| **Hudba** | Icaros, bubny | Klasika, ambient |
| **Integrace** | Komunitni | Individualni terapie |

#### 2. Vyznamovy system

To, jak jedinec interpretuje zkusenost, je ovlivneno jeho vyznamovym systemem:

| Ramec | Interpretace mysticke zkusenosti |
|-------|----------------------------------|
| **Spiritualni** | Setkani s bozstvim, prozreni |
| **Psychologicky** | Pristup k nevedome mysli |
| **Neurovedecky** | Zmeny v mozkovych sitich |
| **Existencialni** | Konfrontace s bytim |

## Interakce set a setting

### Synergicky model

Set a setting neexistuji izolovaně, ale interaguji komplexne:

```
                    SET
                     |
         +-----------+-----------+
         |                       |
    Pozitivni               Negativni
         |                       |
    +----+----+             +----+----+
    |         |             |         |
SETTING    SETTING      SETTING    SETTING
Pozit.    Negat.        Pozit.    Negat.
    |         |             |         |
    v         v             v         v
OPTIMALNI  SMISENA     SMISENA   VYSOKO
ZKUSENOST  ZKUSENOST   ZKUSENOST RIZIKOVA
```

### Predikce vysledku

| Set | Setting | Pravdepodobny vysledek |
|-----|---------|------------------------|
| Pozitivni | Pozitivni | Vysoka sance na mystickou/terapeutickou zkusenost |
| Pozitivni | Negativni | Zvysene riziko obtiznosti, ale mozna zvladnuti |
| Negativni | Pozitivni | Setting muze kompenzovat, ale riziko zustava |
| Negativni | Negativni | Vysoke riziko bad tripu, trauma |

### Duvody spatnych zkusenosti ("Bad trips")

| Faktor | Prispevek k negativni zkusenosti |
|--------|----------------------------------|
| **Nedostatecna priprava** | 30-40% |
| **Nevhodne prostredi** | 20-30% |
| **Vysoka davka** | 15-25% |
| **Psychiatricka vulnerabilita** | 15-20% |
| **Spatny timing** | 10-15% |
| **Lekove interakce** | 5-10% |

## Harm reduction aplikace

### Pred zkusenosti

| Doporuceni | Zduvodneni |
|------------|------------|
| **Psychoedukace** | Realisticka ocekavani, snizeni uzkosti |
| **Screening** | Vylouceni kontraindikaci |
| **Vyber prostredi** | Bezpecne, kontrolovane |
| **Duverhodny pruvodce** | Podpora v obtiznych momentech |
| **Intencni prace** | Jasny zamer |
| **Fyzicka priprava** | Odpocinek, zdravá strava |

### Behem zkusenosti

| Situace | Intervence |
|---------|------------|
| **Uzkost** | Ujisteni, dychani, telesny kontakt |
| **Dezorientace** | Jemne pripomenuti reality |
| **Intenzivni emoce** | Validace, podpora |
| **Somatické potize** | Pohodli, voda, teplota |
| **Psychoza** | Klidne prostredi, SOS benzodiazepiny |

### Po zkusenosti

| Doporuceni | Ucèl |
|------------|------|
| **Integracni terapie** | Zpracovani insights |
| **Komunitni podpora** | Sdileni, normalizace |
| **Kreativni vyjádření** | Artikulace zkusenosti |
| **Casovy odstup** | Minimalne 2-4 tydny mezi sezenimi |

## Klinické protokoly

### Johns Hopkins protocol

Standardni protocol pro klinicke studie s [psilocybinem](/alkaloids/psilocybin/):

| Fáze | Trvání | Aktivity |
|------|--------|----------|
| **Screening** | 1-2 tydny | Psychiatricke vysetreni, vylouceni kontraindikaci |
| **Pripravna sezeni** | 2-4 sezeni | Budovani vztahu, psychoedukace, intence |
| **Davkovaci den** | 6-8 hodin | Podani latky, podpora, monitorace |
| **Integrace** | 2-4 sezeni | Zpracovani zkusenosti, aplikace |
| **Follow-up** | 1-12 mesicu | Dlouhodobe sledovani |

### Prostor pro davkovaci sezeni

| Prvek | Specifikace |
|-------|-------------|
| **Mistnost** | Ticha, pohodlna, domacky vyzdoba |
| **Nabytek** | Pohovka/luzko, kreslá pro terapeuty |
| **Osvetleni** | Regulovatelne, tlumene |
| **Hudba** | Standardizovana playlist, sluchátka |
| **Maska na oci** | Pro internalizaci |
| **WC** | Snadno pristupne |
| **Monitorace** | Krevni tlak, puls (minimalne invazivní) |

### Role terapeuta

| Aspekt | Praxe |
|--------|-------|
| **Priprava** | 8+ hodin pripravnych sezeni |
| **Počet** | 2 terapeuti (muzsko-zenska dyada) |
| **Pristup** | "Trust, let go, be open" |
| **Intervence** | Minimalni, podporujici |
| **Dotyk** | Pouze se souhlasem, rukama drzeni |
| **Verbalni podpora** | "Jste v bezpeci", "Vsechno je v poradku" |

### MAPS MDMA protocol

Protocol pro MDMA-asistovanou psychoterapii PTSD:

| Fáze | Popis |
|------|-------|
| **Priprava** | 3 sezeni (90 min) |
| **Davkovaci sezeni** | 8 hodin, 2-3 celkem |
| **Integrace** | 3 sezeni po kazdem davkovacim |

## Souvisejici koncepty

### Psychedelicky stav

Set and setting moduluji zakladni psychedelicky stav:

- Aktivace [5-HT2A receptoru](/receptors/5-ht2a/)
- Dezintegrace Default Mode Network
- Zvysena entropie mozku
- Neuroplasticita

### Mysticka zkusenost

Optimalni set and setting zvysuji pravdepodobnost:

- Pocit jednoty
- Transcendence casu/prostoru
- Pozitivni nalada
- Noetická kvalita
- Nevyslvitelnost
- Posvátnost
