# Makropsie a mikropsie

**Makropsie** a **mikropsie** jsou poruchy vizuálního vnímání, při kterých se objekty (včetně vlastního těla) zdají neadekvátně **velké** (makropsie) nebo **malé** (mikropsie). Společně s dalšími perceptuálními distorzemi tvoří **Alice in Wonderland syndrome (AIWS)**.

---

## Definice

### Makropsie

| Aspekt | Popis |
|--------|-------|
| **Definice** | Objekty se zdají větší než jsou |
| **Subjektivně** | "Ruce jsou obrovské", "Strop je velmi vysoko" |
| **Typ** | Vizuální nebo somatická |

### Mikropsie

| Aspekt | Popis |
|--------|-------|
| **Definice** | Objekty se zdají menší než jsou |
| **Subjektivně** | "Pokoj je miniaturní", "Lidé jsou jako panenky" |
| **Typ** | Vizuální nebo somatická |

### Telopsie a pelopsie

| Porucha | Popis |
|---------|-------|
| **Telopsie** | Objekty se zdají vzdálenější |
| **Pelopsie** | Objekty se zdají bližší |

---

## Alice in Wonderland Syndrome (AIWS)

### Pojmenování

Syndrom pojmenoval v roce 1955 psychiatr **John Todd** podle románu Lewise Carrolla "Alenka v říši divů", kde hlavní hrdinka zažívá změny velikosti po požití různých substancí.

### Komponenty AIWS

```
ALICE IN WONDERLAND SYNDROME

                    AIWS
                     |
      ┌──────────────┼──────────────┐
      |              |              |
  SOMATICKÉ      VIZUÁLNÍ     TEMPORÁLNÍ
      |              |              |
  Makro/mikro-   Makro/mikro-   Zpomalení/
  somatognozie   psie           zrychlení času
```

| Typ | Podtypy | Příklad |
|-----|---------|---------|
| **Somatické** | Makrosomatognozie | Ruka se zdá obrovská |
|               | Mikrosomatognozie | Nohy se zdají malé |
| **Vizuální** | Makropsie | Pokoj se zdá velký |
|              | Mikropsie | Předměty jsou miniaturní |
| **Temporální** | Dilatace | Čas plyne pomalu |
|                | Kontrakce | Čas plyne rychle |

## Mechanismy

### Neuroanatomie

```
PERCEPČNÍ INTEGRACE

Vizuální kortex → Parietální kortex → Thalamus
      |                 |                |
  Základní           Prostorová      Senzorická
  zpracování         integrace       filtrace
      |                 |                |
      └────────────────┼────────────────┘
                       ↓
              PERCEPČNÍ GESTALT
                       ↓
              Dysfunkce = AIWS
```

### Zapojené struktury

| Oblast | Funkce | Dysfunkce |
|--------|--------|-----------|
| **Parietální kortex** | Prostorové zpracování | Makro/mikropsie |
| **[Thalamus](@/brain/thalamus.md)** | Senzorická filtrace | Distorze velikosti |
| **Temporoparietální junkce** | Tělesné schéma | Somatické změny |

### Farmakologické mechanismy

| Systém | Látky | Mechanismus |
|--------|-------|-------------|
| **GABAergní** | [Muscimol](@/alkaloids/muscimol.md) | Thalamická modulace |
| **Serotoninový** | [Psilocybin](@/alkaloids/psilocybin.md) | Kortikální excitace |
| **Glutamátový** | [Ketamin](@/alkaloids/ketamin.md) | NMDA blokáda |

## Fenomenologie u muscimolu

### Specifické charakteristiky

[Muscimol](@/alkaloids/muscimol.md) je známý vysokou incidencí AIWS symptomů:

```
MUSCIMOL A AIWS

Dávka             Typ symptomů
  |
Nízká ────────── Mírná somatická makro/mikropsie
  |              "Ruce se zdají jiné"
  |
Střední ──────── Výrazná somatická distorze
  |              + mírná vizuální
  |
Vysoká ────────── Komplexní AIWS
                 + oneirické obsahy
```

### Subjektivní popisy

> "Moje ruce vypadaly obrovské, jako ruce obra. Když jsem se díval na nohy, byly naopak malé a vzdálené. Pokoj se zdál jednou velký jako sál, pak malý jako skříň."

### Srovnání s psychedeliky

| Aspekt | Muscimol | LSD/Psilocybin |
|--------|----------|----------------|
| **Frekvence** | Velmi častá | Méně častá |
| **Typ** | Somatická dominance | Vizuální dominance |
| **Charakter** | Statická, perzistentní | Dynamická, proměnlivá |
| **Orientace** | Snížená | Zachovaná |

## Management

### Při intoxikaci

1. **Ujištění** - vysvětlit, že jde o přechodný efekt
2. **Bezpečné prostředí** - riziko dezorientace
3. **Sledování** - může progredovat

### Diferenciální diagnostika

| Stav | Odlišení |
|------|----------|
| **Psychóza** | Zachovaný náhled u AIWS |
| **Migréna** | Anamnéza, bolest hlavy |
| **Epilepsie** | EEG, další iktální jevy |
