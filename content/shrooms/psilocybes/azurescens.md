# Psilocybe azurescens - Lysohlávka azurová

**Psilocybe azurescens** je považována za **nejpotentnější** známý druh [psilocybinových hub](/shrooms/psilocybes/). Byla popsána Paulem Stametsem v roce 1996.

## Taxonomie

| Klasifikace | Hodnota |
|-------------|---------|
| Říše | Fungi |
| Oddělení | Basidiomycota |
| Třída | Agaricomycetes |
| Řád | Agaricales |
| Čeleď | Hymenogastraceae |
| Rod | [Psilocybe](/shrooms/psilocybes/) |
| Druh | P. azurescens |

## Etymologie

- **azurescens** = "modřející" (lat.) - podle intenzivního modření

## Morfologie

### Klobouk
- Průměr: 3-10 cm
- Tvar: kuželovitý → konvexní → plochý s výrazným hrbolem
- Barva: karamelově hnědá → bledě žlutá (suchý)
- Povrch: hladký, lepkavý, s odlučitelnou želatinózní vrstvou

### Třeň
- Výška: 9-20 cm
- Průměr: 3-6 mm
- Barva: bílá → šedohnědá
- Charakter: vláknitý, dutý, často zakřivený
- **Modření**: Velmi intenzivní (indigově modré)

### Lupeny
- Barva: hnědá → tmavě purpurově černá
- Přirůstání: přirostlé k třeni

### Výtrusy
- Barva: tmavě purpurově hnědá
- Rozměry: 12-13,5 × 6,5-8 μm
- Tvar: elipsoidní

## Biotop

### Habitat
- **Ekosystém**: Pobřežní dunové oblasti, písčité půdy
- **Substrát**: Rozpadající se dřevo, štěpky, mulč
- **Asociace**: Ammophila arenaria (písečnice přímořská)

### Geografické rozšíření
- **Původní**: Pacifické pobřeží USA (Oregon, Washington)
- **Oblast**: Columbia River delta, Long Beach Peninsula
- **Klima**: Oceánické, chladné a vlhké

### Sezóna
- **Období**: Září - Leden
- **Optimum**: Říjen - Listopad
- **Podmínky**: Teploty 5-15°C, vysoká vlhkost

## Kultivace

P. azurescens je **obtížně kultivovatelná** v interiéru:

| Faktor | Požadavky |
|--------|-----------|
| Substrát | Dřevěné štěpky (olše, vrba) |
| Teplota | 10-18°C (chladná) |
| Vlhkost | 95%+ |
| Světlo | Nepřímé, nízké |
| Doba kolonizace | 6-12 měsíců |

**Poznámka**: Úspěšná kultivace vyžaduje venkovní záhon s dřevěným mulčem.

## Farmakologie

### Dávkování (POZOR - vysoká potence!)

| Intenzita | Sušené |
|-----------|--------|
| Práh | 0,25 g |
| Mírná | 0,5-1 g |
| Střední | 1-1,5 g |
| Silná | 1,5-2 g |
| Heroická | 2+ g |

**VAROVÁNÍ**: Dávky jsou výrazně nižší než u [P. cubensis](/shrooms/psilocybes/cubensis/)!

### Speciální účinky

P. azurescens je spojována s fenoménem **"dřevěných nohou"** (wood lover's paralysis):
- Dočasná paralýza dolních končetin
- Trvání: 1-4 hodiny
- Pravděpodobná příčina: Neznámý neurotoxin nebo alkaloid
