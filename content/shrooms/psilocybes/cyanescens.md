# Psilocybe cyanescens - Lysohlávka modravá

**Psilocybe cyanescens**, známá jako **Wavy Cap** (vlnitá čepička), je vysoce potentní druh [psilocybinových hub](/shrooms/psilocybes/) s charakteristickým zvlněným okrajem klobouku.

## Taxonomie

| Klasifikace | Hodnota |
|-------------|---------|
| Říše | Fungi |
| Oddělení | Basidiomycota |
| Třída | Agaricomycetes |
| Řád | Agaricales |
| Čeleď | Hymenogastraceae |
| Rod | [Psilocybe](/shrooms/psilocybes/) |
| Druh | P. cyanescens |

## Etymologie

- **cyanescens** = "modřející" (lat. *cyaneus* = modrý)
- **Wavy Cap** = podle vlnitého okraje klobouku

## Morfologie

### Klobouk
- Průměr: 1,5-5 cm
- Tvar: konvexní → plochý s **charakteristicky zvlněným okrajem**
- Barva: karamelově hnědá (vlhký) → bledě žlutá (suchý)
- Povrch: hladký, lesklý, hygrofánní

### Třeň
- Výška: 3-8 cm
- Průměr: 3-6 mm
- Barva: bílá → šedohnědá od báze
- Charakter: vláknitý, křehký
- **Modření**: Silné, indigově modré

### Lupeny
- Barva: bledě hnědá → tmavě purpurově hnědá
- Přirůstání: přirostlé

### Výtrusy
- Barva: tmavě purpurově hnědá
- Rozměry: 9-12 × 5-8 μm
- Tvar: elipsoidní až subfusiformní

## Aktivní látky

| Alkaloid | Koncentrace (suchá hmotnost) |
|----------|------------------------------|
| [Psilocybin](/alkaloids/psilocybin/) | 0,39-0,66% |
| [Psilocin](/alkaloids/psilocin/) | 0,14-0,30% |
| Baeocystin | 0,02-0,03% |

**Celková potence**: ~0,85% (vyšší než [P. cubensis](/shrooms/psilocybes/cubensis/))

## Biotop

### Habitat
- **Ekosystém**: Městské parky, zahrady, lesní okraje
- **Substrát**: Dřevěné štěpky, mulč, piliny
- **Preference**: Rozpadající se listnaté dřevo

### Geografické rozšíření

| Region | Status |
|--------|--------|
| Západní Evropa | Rozšířený |
| Střední Evropa | Přítomný |
| Pacifické pobřeží USA | Velmi hojný |
| Austrálie/Nový Zéland | Introdukovaný |

**Rozšiřování**: P. cyanescens se aktivně šíří s přepravou zahradního mulče.

### Sezóna
- **Období**: Říjen - Prosinec
- **Optimum**: Listopad (po prvních mrazech)
- **Podmínky**: Teploty 2-13°C, vysoká vlhkost

## Ekologie

### Wood Lover (Lignikolní)

P. cyanescens je striktně **lignikolní** (vázaná na dřevo):
- Preferuje štěpky z listnatých stromů
- Olše, vrba, buk, dub
- Neroste přímo na půdě

### Invazivní potenciál

Druh se šíří s:
- Zahradním mulčem
- Přesunem kontaminované zeminy
- Mezinárodním obchodem s rostlinami

## Kultivace

| Faktor | Požadavky |
|--------|-----------|
| Substrát | Dřevěné štěpky (listnaté) |
| Teplota inkubace | 15-20°C |
| Teplota fruktifikace | 7-13°C (chlad nutný!) |
| Vlhkost | 85-95% |
| Světlo | Nepřímé denní |

**Poznámka**: Fruktifikace vyžaduje **teplotní šok** (pokles pod 10°C).

## Farmakologie

### Dávkování

| Intenzita | Sušené |
|-----------|--------|
| Práh | 0,5 g |
| Mírná | 1-1,5 g |
| Střední | 1,5-2,5 g |
| Silná | 2,5-3,5 g |
| Heroická | 3,5+ g |

### Wood Lover's Paralysis

Podobně jako [P. azurescens](/shrooms/psilocybes/azurescens/), i P. cyanescens může způsobit:
- Dočasnou svalovou slabost
- Necitlivost končetin
- Trvání: 1-6 hodin

| Znak | P. cyanescens | Galerina marginata |

| Výtrusy | Purpurově hnědé | Rezavě hnědé |
| Jedovatost | Nejedovatá | **SMRTELNÁ** |

## Historie

- **1946**: Popsána Elsie Wakefield (Kew Gardens)
- **1960s**: Rozpoznána jako psilocybinová
- **1990s**: Masivní rozšíření v městských oblastech
