# Fencyklidin - Disociativni anestetikum

**Fencyklidin** (PCP, "Angel Dust") je synteticky **arylcyklohexylamin** puvodne vyvinuty jako intravenn anestetikum. Je prototypickym [NMDA receptorovym](/receptors/nmda/) antagonistou s komplexnim multi-receptorovym profilem. Klinicky opusten kvuli vyznamne dysforii a emergencnim reakcim, ale zostava dulezitou slouceninou pro vyzkum glutamatergni dysfunkce u schizofrenie. Prekurzor modernich disociativ vcetne [ketaminu](/alkaloids/ketamin/).

PCP MODEL SCHIZOFRENIE

       - PCP intoxikace pripomina

### Prodlouzeny polocas - dulezita charakteristika

```
FARMAKOKINETICKY PROFIL PCP

Polocas: 7-46 hodin (prumer ~21 h)
         vs Ketamin: 2-3 hodiny

Duvody:
1. Vysoká lipofilita (logP 4,6)
   - Redistribuce do tukové tkáně
   - Pomalé uvolnování

2. Enterohepatalni recirkulace
   - Opakovaná absorpce
   - Prolongované účinky

3. Aktivní metabolity
   - PCP-OH (aktivní)
   - Další hydroxylované metabolity

Klinický důsledek:
- Prolongované intoxikace (dny)
- Fluktuující symptomy
- Náročná detoxifikace
```

## Toxikologie a bezpecnost

### Klinicky obraz intoxikace

```
PCP INTOXIKACE - KLINICKÉ PŘÍZNAKY

TRIÁDA:
1. Nystagmus (vertikální nebo horizontální)
2. Hypertenze
3. Abnormální chování (agitace nebo stupor)

NEUROLOGICKÉ:
- Nystagmus (charakteristický!)
- Ataxie
- Dysartrie
- Svalová rigidita
- Myoklonus
- Křeče (vysoké dávky)
- Hypereflexie

PSYCHIATRICKÉ:
- Agitace/agresivita
- Bizarní chování
- Halucinace
- Paranoia
- Katatonie
- Disociace

AUTONOMNÍ:
- Hypertenze
- Tachykardie
- Hyperthermie
- Mydriáza
- Hypersalivace
- Diaphoréza

ZÁVAŽNÉ (vysoké dávky):
- Kóma
- Status epilepticus
- Rhabdomyolyza
- Akutní renální selhání
- Respirační selhání
```

### Diferencialni diagnostika

| Stav | Odlisujici znak |
|------|-----------------|
| **Anticholinergni syndrom** | Sucha kuze, retence moci, mydriaza |
| **Sympatomimetická intoxikace** | Podobne, ale bez nystagmu |
| **Serotonin syndrom** | Hyperreflexie, klonus |
| **Alkohol intoxikace** | Zadny nystagmus, mene agitace |
| **Ketamin** | Kratsi trvani, mene agresivity |

#### Emergenční agitace

PCP je znamy pro extremni agitaci a agresi:

| Faktor | Popis |
|--------|-------|
| **Mechanismus** | NMDA blokada + dopaminergni aktivace |
| **Projev** | "Superhuman" síla (mýtus, ale zvýšená odolnost k bolesti) |
| **Nebezpečí** | Násilí vůči sobě i okolí |
| **Management** | Bezpečnostní opatření, sedace |

### Lécba intoxikace

| Intervence | Indikace | Poznámka |
|------------|----------|----------|
| **Klidné prostredí** | Vždy | Minimální stimulace |
| **Benzodiazepiny** | Agitace, křeče | Lorazepam 2-4 mg IV |
